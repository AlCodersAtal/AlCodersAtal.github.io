How CSS Works

CSS breathes life into HTML. Learn the three ways to add CSS and why one method is far better than the others.

3 Ways to Add CSS

CSS Methods
/* 1. Inline — directly on the element (avoid this) */ <p style="color:red;">Red text</p> /* 2. Internal — inside a <style> tag in <head> */ <style> p { color: blue; } </style> /* 3. External — separate .css file (BEST) */ <link rel="stylesheet" href="style.css">
💡 Always use External CSS
Keep your CSS in a separate style.css file. It keeps your HTML clean and lets you reuse styles across multiple pages.

CSS Syntax

CSS Rule
selector { property: value; property: value; } /* Example */ h1 { color: darkblue; font-size: 2rem; }

CSS Selectors

Selectors let you target specific HTML elements to apply styles. Master these and you can style anything.

Types of Selectors

Selectors
/* Element selector — targets all <p> tags */ p { color: gray; } /* Class selector — targets class="intro" */ .intro { font-size: 1.2rem; } /* ID selector — targets id="hero" */ #hero { background: lightblue; } /* Descendant — <p> inside .card */ .card p { margin: 0; } /* Multiple selectors */ h1, h2, h3 { font-family: Georgia; }
ℹ️ Class vs ID
Use classes (.name) for multiple elements. Use IDs (#name) for unique elements (only one per page).

Colors & Backgrounds

Colour is one of the most powerful design tools you have. Learn every way CSS lets you define and apply colour.

Ways to Define Color

Colors
p { color: red; } /* Named */ p { color: #ff0000; } /* Hex */ p { color: rgb(255,0,0); } /* RGB */ p { color: rgba(255,0,0,0.5); }/* RGBA — with opacity */ p { color: hsl(0,100%,50%); } /* HSL */

Backgrounds

Backgrounds
div { background-color: lightblue; background-image: url("bg.jpg"); background-size: cover; background-position: center; background-repeat: no-repeat; } /* Gradient background */ div { background: linear-gradient(135deg, #1a5cff, #ff5c2b); }

Typography

Typography is the art of arranging text. Good typography makes pages readable, beautiful, and professional.

Font Properties

Typography
p { font-family: Georgia, serif; font-size: 1rem; /* 1rem = 16px */ font-weight: bold; /* normal, bold, 100–900 */ font-style: italic; line-height: 1.6; letter-spacing: 0.5px; text-align: center; /* left, right, center, justify */ text-transform: uppercase; /* lowercase, capitalize */ text-decoration: underline; color: #333; }

Using Google Fonts

Google Fonts
<!-- In <head> --> <link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;700&display=swap" rel="stylesheet"> /* In CSS */ body { font-family: 'Poppins', sans-serif; }

The Box Model

Every element on a webpage is a box. The box model describes the space around each element — and mastering it unlocks layout control.

The 4 Layers of the Box Model

Box Model
div { /* Content size */ width: 300px; height: 150px; /* Padding — space INSIDE, between content and border */ padding: 20px; padding: 10px 20px; /* top/bottom left/right */ padding: 5px 10px 15px 20px; /* top right bottom left */ /* Border — the visible edge */ border: 2px solid #333; border-radius: 8px; /* rounded corners */ /* Margin — space OUTSIDE, between this and other elements */ margin: 20px; }
  • Content — the actual text or image inside
  • Padding — space inside, between content and border
  • Border — the visible outline
  • Margin — space outside, separating from other elements
