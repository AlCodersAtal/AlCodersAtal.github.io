Headings & Paragraphs

Text is the most fundamental part of any webpage. Learn to structure it with headings and paragraphs.

The 6 Heading Levels

HTML has six heading tags from <h1> (biggest) to <h6> (smallest). Think of them like chapter titles and subtitles in a book.

Headings
<h1>Main Page Title</h1> <h2>Section Heading</h2> <h3>Sub-section Heading</h3> <h4>Smaller Sub-heading</h4> <h5>Even Smaller</h5> <h6>Smallest Heading</h6>
⚠️ Use only one <h1> per page
It's your main title. Use h2 for sections, h3 for sub-sections. This helps search engines and screen readers.

Paragraphs & Text Formatting

Text Tags
<p>This is a paragraph of text.</p> <p>HTML is <strong>very important</strong> to learn.</p> <p>The word <em>semantics</em> means "meaning".</p> <p>Line one.<br>Line two, same paragraph.</p>
💡 <strong> vs <em>
Use <strong> for important bold text, <em> for emphasized italic text. They're better than <b> and <i> because they carry meaning.

Links & Anchors

Links are what makes the web a web. Learn how to connect pages, link to external sites, and jump to sections.

The <a> Tag

Links are created with the <a> (anchor) tag. The href attribute tells the browser where to go:

Links
<!-- External link --> <a href="https://google.com">Go to Google</a> <!-- Link to another page in your site --> <a href="about.html">About Me</a> <!-- Open in a new tab --> <a href="https://github.com" target="_blank">GitHub</a> <!-- Jump to a section on the same page --> <a href="#contact">Jump to Contact</a> <section id="contact">...</section>

Email and Phone Links

Special Links
<!-- Opens email client --> <a href="mailto:hello@example.com">Email Me</a> <!-- Calls on mobile --> <a href="tel:+1234567890">Call Me</a>
💡 Link as an Image
You can wrap anything in an <a> tag — including images — to make it clickable!

Images

Images make websites come alive. Learn to add, resize, and describe images using the <img> tag.

The <img> Tag

Images use the self-closing <img> tag with two essential attributes:

Images
<!-- Image from your project folder --> <img src="photo.jpg" alt="A description of the photo"> <!-- Image with size --> <img src="logo.png" alt="Logo" width="200" height="100"> <!-- Image from the internet --> <img src="https://placekitten.com/300/200" alt="A kitten">
  • src — the path or URL to the image file
  • alt — a text description (shown if image fails to load; used by screen readers)
⚠️ Always use alt text
The alt attribute is important for accessibility and SEO. Never skip it!

Image File Paths

File Paths
<!-- Same folder --> <img src="photo.jpg" alt="..."> <!-- Inside an "images" folder --> <img src="images/photo.jpg" alt="..."> <!-- Go up one folder, then into images --> <img src="../images/photo.jpg" alt="...">

Lists — Ordered & Unordered

Lists are everywhere on the web — navigation menus, ingredients, steps, features. Learn both types.

Unordered Lists — Bullet Points

Unordered List
<ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul>

Ordered Lists — Numbered

Ordered List
<ol> <li>Learn HTML</li> <li>Learn CSS</li> <li>Build a website</li> </ol>

Nested Lists

Nested
<ul> <li>Frontend <ul> <li>HTML</li> <li>CSS</li> </ul> </li> <li>Backend</li> </ul>

Tables

Tables organise data into rows and columns. Perfect for schedules, comparison charts, and structured data.

Table Structure

Basic Table
<table> <!-- Table header row --> <thead> <tr> <th>Name</th> <th>Age</th> <th>City</th> </tr> </thead> <!-- Table body rows --> <tbody> <tr> <td>Ali</td> <td>22</td> <td>Dubai</td> </tr> <tr> <td>Sara</td> <td>25</td> <td>London</td> </tr> </tbody> </table>
  • <table> — the container
  • <thead> — header section
  • <tbody> — body section
  • <tr> — table row
  • <th> — header cell (bold & centered by default)
  • <td> — data cell
