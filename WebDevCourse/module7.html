CSS Transitions

Transitions add smooth animation between CSS states — like hovering over a button.

The transition Property

Transitions
.btn { background: #1a5cff; color: white; padding: 12px 24px; /* property duration timing-function */ transition: background 0.3s ease; transition: all 0.3s ease; /* animate everything */ } .btn:hover { background: #0047e0; transform: translateY(-2px); }

CSS Transforms

Transforms let you move, scale, rotate, and skew elements — without affecting the layout around them.

Transform Functions

Transforms
.box { transform: translateX(50px); /* move right */ transform: translateY(-20px); /* move up */ transform: translate(20px, 10px);/* move right+down */ transform: scale(1.5); /* 150% bigger */ transform: rotate(45deg); /* rotate 45° */ transform: skewX(10deg); /* skew horizontally */ /* Combine multiple */ transform: translateY(-4px) scale(1.05); }

CSS Animations with @keyframes

@keyframes lets you create multi-step animations — fades, bounces, pulses, and more.

@keyframes Syntax

@keyframes
/* Define the animation */ @keyframes fadeIn { from { opacity: 0; transform: translateY(20px); } to { opacity: 1; transform: translateY(0); } } @keyframes pulse { 0% { transform: scale(1); } 50% { transform: scale(1.1); } 100% { transform: scale(1); } } /* Apply the animation */ .hero-text { animation: fadeIn 0.6s ease forwards; /* name duration timing fill-mode */ } .badge { animation: pulse 2s ease infinite; /* loops forever */ }

Hover & Focus Effects

Pseudo-classes like :hover, :focus, and :active make your interface respond to user actions.

CSS Pseudo-Classes

Pseudo-classes
/* When mouse is over element */ .btn:hover { background: darkblue; } /* When element is focused (clicked/tabbed to) */ input:focus { outline: none; border-color: #1a5cff; box-shadow: 0 0 0 3px rgba(26,92,255,0.2); } /* When element is being clicked */ .btn:active { transform: scale(0.97); } /* Visited / unvisited links */ a:visited { color: purple; } a:link { color: blue; }
