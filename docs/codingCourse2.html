Module 1: Python Basics & Variables

Why Python?

Python is beginner-friendly, versatile, and widely used in AI, web dev, and automation.

Setting Up

Install Python and an IDE like VS Code. Learn to run your first Python script.

Variables & Data Types

Variables temporarily store data in memory.

price = 10
rating = 4.9
course_name = 'Python for Beginners'
is_published = True

Explanation:

  • price is an integer (whole number)
  • rating is a float (decimal number)
  • course_name is a string (text)
  • is_published is a boolean (True/False)

Comments

Comments add notes in code, ignored by Python.

# This is a comment and won't run
# Comments can be multiple lines

Module 2: User Input & Strings

Receiving Input

Use input() to get user input (always returns string).

birth_year = int(input('Birth year: '))  # convert to integer

Strings

Strings hold text data. Use single ('') or double ("") quotes.

Multi-line strings use triple quotes (""" """).

Accessing Characters & Slicing

course = 'Python for Beginners'
print(course[0])   # 'P'
print(course[-1])  # 's'
print(course[1:5]) # 'ytho'

String Methods & Formatting

name = 'Mosh'
message = f'Hi, my name is {name}'

print(message.upper())   # uppercase
print(message.lower())   # lowercase
print(message.title())   # capitalize each word
print(message.find('p')) # find index of 'p' (-1 if not found)
print(message.replace('p', 'q')) # replace 'p' with 'q'

contains = 'Python' in course  # True/False check

Module 3: Operators and Conditionals

Arithmetic Operators

x = 10
y = 3
print(x + y)   # 13
print(x - y)   # 7
print(x * y)   # 30
print(x / y)   # 3.3333 (float division)
print(x // y)  # 3 (integer division)
print(x % y)   # 1 (remainder)
print(x ** y)  # 1000 (exponentiation)

Augmented Assignment

x += 10  # equivalent to x = x + 10

Operator Precedence

Order of operations:

  1. Parentheses
  2. Exponentiation
  3. Multiplication & Division
  4. Addition & Subtraction

If Statements & Logical Operators

is_hot = True
is_cold = False

if is_hot:
    print("Hot day")
elif is_cold:
    print("Cold day")
else:
    print("Beautiful day")

has_high_income = True
has_good_credit = False

if has_high_income and has_good_credit:
    print("Eligible for loan")

if has_high_income or has_good_credit:
    print("Considered for loan")

is_day = True
is_night = not is_day

Comparison Operators

a = 5
b = 10

print(a > b)   # False
print(a >= b)  # False
print(a < b)   # True
print(a <= b)  # True
print(a == b)  # False
print(a != b)  # True

Module 4: Loops

While Loops

i = 1
while i < 5:
    print(i)
    i += 1

For Loops and Range

for i in range(1, 5):
    print(i)

# range(5) generates 0,1,2,3,4
# range(1,5) generates 1,2,3,4
# range(1,5,2) generates 1,3

Module 5: Data Structures — Lists, Tuples & Dictionaries

Lists

numbers = [1, 2, 3, 4, 5]
print(numbers[0])  # 1
print(numbers[-1]) # 5

numbers.append(6)
numbers.insert(0, 6)
numbers.remove(6)
numbers.pop()
numbers.clear()

print(numbers.index(3))
numbers.sort()
numbers.reverse()
numbers_copy = numbers.copy()

Tuples

Immutable lists.

coordinates = (1, 2, 3)
x, y, z = coordinates  # unpacking

Dictionaries

customer = {
    "name": "John Smith",
    "age": 30,
    "is_verified": True
}
print(customer["name"])          # John Smith
print(customer.get("type", "silver"))  # silver (default if key not found)

customer["name"] = "New Name"

Module 6: Functions

Defining Functions

def greet_user(name):
    print(f"Hi {name}")

greet_user("John")

Arguments

Positional and keyword arguments

def calculate_total(order, shipping=5, tax=0.1):
    total = order + shipping + (order * tax)
    return total

print(calculate_total(order=50))

Return Values

def square(number):
    return number * number

result = square(2)
print(result)  # 4

Module 7: Exceptions

Handle errors gracefully to prevent crashes.

try:
    age = int(input('Age: '))
    income = 20000
    risk = income / age
    print(age)
except ValueError:
    print('Not a valid number')
except ZeroDivisionError:
    print('Age cannot be 0')

Module 8: Classes and Inheritance

Classes

class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def move(self):
        print("move")

point1 = Point(10, 5)
point2 = Point(2, 4)

Inheritance

class Mammal:
    def walk(self):
        print("walk")

class Dog(Mammal):
    def bark(self):
        print("bark")

dog = Dog()
dog.walk()
dog.bark()

Module 9: Modules and Packages

Modules

# Import entire module
import converters
converters.kg_to_lbs(5)

# Import specific function
from converters import kg_to_lbs
kg_to_lbs(5)

Packages

# Import entire module from package
from ecommerce import sales
sales.calc_shipping()

# Import specific function from package
from ecommerce.sales import calc_shipping
calc_shipping()

Module 10: Python Standard Library & PyPI

Standard Library — Random Module

import random

print(random.random())        # float between 0 and 1
print(random.randint(1, 6))   # integer between 1 and 6

members = ['John', 'Bob', 'Mary']
leader = random.choice(members)  # randomly pick an item

PyPI and Pip

PyPI is a directory of Python packages. Use pip to install/uninstall packages.

pip install openpyxl
pip uninstall openpyxl
