Code Reference · Level 1

Basic On/Off

Simple threshold switch — the first sketch every LDR project starts with.

📋
Copy-Ready

All code is ready to paste into Arduino IDE. Upload speed: 9600 baud for Serial Monitor.

Arduino ldr_basic.ino
// LDR Controlled LED — Basic Version // Connect LDR to A0, LED + 220Ω to pin 13 const int ldrPin = A0; const int ledPin = 13; const int threshold = 500; // Adjust to taste void setup() { pinMode(ledPin, OUTPUT); Serial.begin(9600); } void loop() { int ldrValue = analogRead(ldrPin); Serial.print("LDR: "); Serial.println(ldrValue); if (ldrValue < threshold) { digitalWrite(ledPin, HIGH); // Dark → LED ON } else { digitalWrite(ledPin, LOW); // Bright → LED OFF } delay(100); }
Code Reference · Level 2

Smooth PWM Dimming

Analog brightness control — LED gets brighter as it gets darker using map().

Arduino ldr_pwm.ino
// LDR + LED with smooth PWM dimming // LED gets brighter as it gets darker // Move LED from pin 13 → pin 9 (PWM ~) const int ldrPin = A0; const int ledPin = 9; // PWM pin required void setup() { pinMode(ledPin, OUTPUT); Serial.begin(9600); } void loop() { int ldrValue = analogRead(ldrPin); int brightness = map(ldrValue, 0, 1023, 255, 0); brightness = constrain(brightness, 0, 255); analogWrite(ledPin, brightness); Serial.print("LDR="); Serial.print(ldrValue); Serial.print(" | LED="); Serial.println(brightness); delay(50); }

3-Zone Visual Gauge

Three LEDs act as a light-level indicator — green for bright, yellow for dim, red for dark.

Arduino ldr_meter.ino
// 3-LED light meter // Green = bright | Yellow = dim | Red = dark const int ldrPin = A0; const int ledGreen = 11; const int ledYellow = 10; const int ledRed = 9; void setup() { pinMode(ledGreen, OUTPUT); pinMode(ledYellow, OUTPUT); pinMode(ledRed, OUTPUT); Serial.begin(9600); } void loop() { int val = analogRead(ldrPin); Serial.println(val); digitalWrite(ledGreen, val > 700); digitalWrite(ledYellow, (val >= 300) && (val <= 700)); digitalWrite(ledRed, val < 300); delay(100); }
