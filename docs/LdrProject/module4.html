What is PWM?

Pulse-Width Modulation lets you simulate analog output on digital pins — the key to smooth LED dimming.

💡
PWM in a Nutshell

PWM rapidly switches a pin ON/OFF. The percentage of time it's ON (duty cycle) controls perceived brightness. 0% = off, 100% = full brightness. analogWrite() accepts 0–255.

~
PWM Pins on UnoOnly pins marked ~ support analogWrite(): 3, 5, 6, 9, 10, 11. Move your LED from pin 13 to pin 9.
↔
map() Functionmap(value, fromLow, fromHigh, toLow, toHigh) scales one range to another. Perfect for LDR → brightness.
✅
constrain() Functionconstrain(x, min, max) clamps a value within bounds — prevents values going outside 0–255.
Coding · Module 4

PWM Code & map()

Use analogWrite() and map() to smoothly dim your LED in inverse proportion to ambient light.

Arduino module4_pwm.ino
// Module 4 — Smooth PWM LED Brightness // Move LED wire from pin 13 to pin 9 (~) const int ldrPin = A0; const int ledPin = 9; // Must be a PWM pin ~ void setup() { pinMode(ledPin, OUTPUT); Serial.begin(9600); } void loop() { int ldrValue = analogRead(ldrPin); // Invert: dark (low LDR value) = bright LED int brightness = map(ldrValue, 0, 1023, 255, 0); brightness = constrain(brightness, 0, 255); analogWrite(ledPin, brightness); Serial.print("LDR="); Serial.print(ldrValue); Serial.print(" Brightness="); Serial.println(brightness); delay(50); // Faster loop for smooth dimming }
How map() Works Here
→
ldrValue = 0 (pitch dark)map(0, 0, 1023, 255, 0) → brightness = 255 → LED fully ON
→
ldrValue = 512 (half light)map(512, 0, 1023, 255, 0) → brightness ≈ 128 → LED at half brightness
→
ldrValue = 1023 (full light)map(1023, 0, 1023, 255, 0) → brightness = 0 → LED fully OFF
