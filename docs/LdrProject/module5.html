Light Meter Zones

Control three LEDs to create a visual light-level meter — each LED represents a lighting zone.

3-Zone Light Indicator
Green LEDBright light
LDR > 700
Yellow LEDMedium light
LDR 300–700
Red LEDDark / dim
LDR < 300
Wiring the Extra LEDs
G
Green LED → Pin 11Anode through 220Ω to pin 11, cathode to GND.
Y
Yellow LED → Pin 10Anode through 220Ω to pin 10, cathode to GND.
R
Red LED → Pin 9Anode through 220Ω to pin 9, cathode to GND.

3-LED Code

Clean, elegant Arduino code using boolean comparisons directly in digitalWrite() calls.

Arduino module5_meter.ino
// Module 5 — 3-LED Light Meter const int ldrPin = A0; const int ledGreen = 11; // Bright const int ledYellow = 10; // Medium const int ledRed = 9; // Dark void setup() { pinMode(ledGreen, OUTPUT); pinMode(ledYellow, OUTPUT); pinMode(ledRed, OUTPUT); Serial.begin(9600); } void loop() { int val = analogRead(ldrPin); Serial.println(val); digitalWrite(ledGreen, val > 700); digitalWrite(ledYellow, (val >= 300) && (val <= 700)); digitalWrite(ledRed, val < 300); delay(100); }
✨
Clean Code Trick

In Arduino, true = HIGH = 1 and false = LOW = 0. Passing a boolean comparison directly to digitalWrite() is perfectly valid — and much more readable than an if/else chain.
