Project: Auto Night Lamp

This final project combines everything: voltage divider wiring, analogRead, PWM brightness, debounce filtering, and adjustable threshold using a potentiometer.

Extra Components Needed
+
10kΩ PotentiometerWiper to A1 pin. Used to set the darkness threshold dynamically without re-uploading code.
+
100µF Capacitor (optional)In parallel with LDR to smooth out rapid fluctuations from flickering lights or electrical noise.
What Makes This Production-Quality?
Quality Checklist
  1. Smoothed Readings — 10-sample averaging eliminates noise spikes from fluorescent lights and electrical interference.
  2. Debounce Filter — Prevents rapid ON/OFF flicker when the LDR value hovers right at the threshold boundary.
  3. Adjustable Threshold — Potentiometer allows threshold adjustment (200–800) without modifying or re-uploading code.
  4. Proportional Brightness — LED dims smoothly based on how dark it is — brighter in deep dark, dimmer near the threshold.

Full Project Code

The complete night_lamp_final.ino — copy, upload, and you're done.

Arduino night_lamp_final.ino
// =========================================== // AUTO NIGHT LAMP — Final Project // Features: PWM dimming, adjustable threshold, // debounce filter, Serial Monitor output // =========================================== const int ldrPin = A0; // LDR voltage divider const int potPin = A1; // Potentiometer for threshold const int ledPin = 9; // PWM LED pin const int SAMPLES = 10; // Readings to average const int DEBOUNCE_MS = 300; // State debounce time bool ledState = false; unsigned long lastChange = 0; void setup() { pinMode(ledPin, OUTPUT); Serial.begin(9600); Serial.println("Night Lamp Ready"); } // Average multiple readings to reduce noise int smoothRead(int pin) { long sum = 0; for (int i = 0; i < SAMPLES; i++) { sum += analogRead(pin); delay(2); } return sum / SAMPLES; } void loop() { int ldrValue = smoothRead(ldrPin); int potValue = analogRead(potPin); int threshold = map(potValue, 0, 1023, 200, 800); bool shouldBeOn = (ldrValue < threshold); // Debounce: only change state after DEBOUNCE_MS if (shouldBeOn != ledState) { if (millis() - lastChange > DEBOUNCE_MS) { ledState = shouldBeOn; lastChange = millis(); } } if (ledState) { int brightness = map(ldrValue, 0, threshold, 255, 50); brightness = constrain(brightness, 0, 255); analogWrite(ledPin, brightness); } else { analogWrite(ledPin, 0); } Serial.print("LDR:"); Serial.print(ldrValue); Serial.print(" Thr:"); Serial.print(threshold); Serial.print(" LED:"); Serial.println(ledState ? "ON" : "OFF"); }
