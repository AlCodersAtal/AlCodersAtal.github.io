Course Quiz

Test your understanding of LDR circuits, Arduino coding, and circuit design. Score 4/4 to complete the course!

1. When ambient light increases, what happens to an LDR's resistance?
✓ Correct! LDRs are photoresistors — more photons = more free electrons = lower resistance. In the dark it can reach 1MΩ; in bright sunlight, just 1kΩ.
2. Which Arduino function reads an analog pin and returns a value between 0 and 1023?
✓ Correct! analogRead() uses the 10-bit ADC to return 0–1023 from pins A0–A5. 0 = 0V, 1023 = 5V.
3. Why is a 10kΩ resistor placed in series with the LDR?
✓ Correct! The LDR + 10kΩ create a voltage divider. As LDR resistance changes with light, the midpoint voltage changes, which analogRead() measures.
4. Which pins on Arduino Uno support analogWrite() for PWM output?
✓ Correct! PWM is only available on the 6 pins marked with ~. Pin 13 (built-in LED) is NOT a PWM pin — always use pin 9, 10, or 11 for LED dimming.
