analogRead() Explained

Understand the 10-bit ADC, voltage-to-number conversion, and how to set a useful darkness threshold.

💡
What Does analogRead() Return?

analogRead(A0) returns a value from 0 to 1023 — 0 means 0V, 1023 means 5V. The LDR voltage divider converts light intensity into this range.

1
10-Bit ADCArduino has a 10-bit Analog-to-Digital Converter. 2¹⁰ = 1024 steps → values 0–1023.
2
Bright light → High valueLDR resistance drops → voltage at A0 rises → analogRead returns ~900–1023.
3
Dark → Low valueLDR resistance rises → voltage at A0 drops → analogRead returns ~0–200.
4
Setting a ThresholdPick a value (e.g. 500) — below this = dark → turn LED ON. Above = bright → LED OFF.

Threshold Code

Write your first LDR sketch — read the sensor, compare to threshold, switch the LED.

Basic Threshold Sketch
Arduino module3_threshold.ino
// Module 3 — Analog Read & Threshold const int ldrPin = A0; const int ledPin = 13; const int threshold = 500; // Adjust based on your environment void setup() { pinMode(ledPin, OUTPUT); Serial.begin(9600); Serial.println("LDR Monitor started..."); } void loop() { int ldrValue = analogRead(ldrPin); Serial.print("LDR Value: "); Serial.print(ldrValue); Serial.print(" | Status: "); if (ldrValue < threshold) { digitalWrite(ledPin, HIGH); Serial.println("DARK - LED ON"); } else { digitalWrite(ledPin, LOW); Serial.println("BRIGHT - LED OFF"); } delay(200); // Read every 200ms }
⚠️
Calibration Tip

Open the Serial Monitor (Ctrl+Shift+M) at 9600 baud. Cover the LDR with your hand and note the value — that's your "dark" reading. Use the midpoint between bright and dark as your threshold.
