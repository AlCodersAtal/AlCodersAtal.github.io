Voltage Divider

Arduino reads voltage, not resistance — learn how the LDR + 10kΩ pair converts light into a readable number.

💡
Why a Voltage Divider?

Arduino's analogRead() measures voltage (0–5V), not resistance. The LDR + 10kΩ resistor form a voltage divider — as light changes LDR resistance, the midpoint voltage changes and analogRead() captures it.

How It Works — Step by Step
1
10-Bit ADCArduino has a 10-bit Analog-to-Digital Converter. 2¹⁰ = 1024 steps → values 0–1023.
2
Bright light → High valueLDR resistance drops → voltage at A0 rises → analogRead returns ~900–1023.
3
Dark → Low valueLDR resistance rises → voltage at A0 drops → analogRead returns ~0–200.
4
ThresholdPick a value (e.g. 500) — below this = dark → turn LED ON. Above = bright → LED OFF.

Safety Check

Always include the 220Ω resistor in series with the LED. Connecting an LED directly to a 5V pin without a current-limiting resistor will burn it out instantly.
