DC Motors Explained

DC motors convert electrical energy into rotational motion — the muscle of your robot.

When current flows through the motor's coil, it creates a magnetic field that spins the rotor. Reverse the current direction → motor reverses. This is how your robot moves forward and backward.

TT Gear Motors

SpecTypical Value
Operating Voltage3V – 6V
No-load Speed~200 RPM at 5V
Stall Current~250–400mA

🚫 NEVER Connect Motors Directly to Arduino
A TT motor draws 200–400mA when loaded. Arduino pins supply only 40mA. Direct connection will immediately destroy the Arduino. Always use a motor driver.

L298N Motor Driver

The L298N dual H-bridge takes small logic signals from Arduino and switches enough current to power two motors independently.

L298N PinFunctionConnect To
12VMotor power inputBattery positive (+)
GNDCommon groundBattery (−) AND Arduino GND
5V outRegulated 5V outputArduino 5V pin
IN1 / IN2Motor A directionArduino Pins 2, 3
IN3 / IN4Motor B directionArduino Pins 4, 5
ENAMotor A speed (PWM)Arduino Pin 9 (~)
ENBMotor B speed (PWM)Arduino Pin 10 (~)
OUT1 / OUT2Motor A terminalsLeft motor wires
OUT3 / OUT4Motor B terminalsRight motor wires
ℹ️ ENA/ENB Jumpers
By default, jumper caps connect ENA/ENB to 5V — full speed always. Remove the jumpers and connect ENA/ENB to Arduino PWM pins to enable variable speed control.

Module 06 · Lesson 3

Speed Control with PWM

analogWrite(pin, 0–255) controls motor speed by rapidly switching power on and off — the ratio determines speed.

Arduino C++ — Complete Motor Library
#define IN1 2 #define IN2 3 #define ENA 9 #define IN3 4 #define IN4 5 #define ENB 10 void driveForward(int s) { analogWrite(ENA,s); analogWrite(ENB,s); digitalWrite(IN1,HIGH); digitalWrite(IN2,LOW); digitalWrite(IN3,HIGH); digitalWrite(IN4,LOW); } void driveBackward(int s) { analogWrite(ENA,s); analogWrite(ENB,s); digitalWrite(IN1,LOW); digitalWrite(IN2,HIGH); digitalWrite(IN3,LOW); digitalWrite(IN4,HIGH); } void spinLeft(int s) { analogWrite(ENA,s); analogWrite(ENB,s); digitalWrite(IN1,LOW); digitalWrite(IN2,HIGH); digitalWrite(IN3,HIGH); digitalWrite(IN4,LOW); } void spinRight(int s) { analogWrite(ENA,s); analogWrite(ENB,s); digitalWrite(IN1,HIGH); digitalWrite(IN2,LOW); digitalWrite(IN3,LOW); digitalWrite(IN4,HIGH); } void stopMotors() { analogWrite(ENA,0); analogWrite(ENB,0); }

Servo Motors

Servo motors rotate to a precise angle — perfect for panning a sensor or steering.

Arduino C++ with Servo.h
#include <Servo.h> Servo scanServo; void setup() { scanServo.attach(6); // Signal wire → pin 6 scanServo.write(90); // Start centred } void loop() { scanServo.write(10); delay(500); // Look left scanServo.write(90); delay(500); // Centre scanServo.write(170); delay(500); // Look right }
