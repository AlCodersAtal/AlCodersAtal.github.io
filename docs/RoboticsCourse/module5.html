Ultrasonic Distance Sensor (HC-SR04)

The HC-SR04 is the robot's eyes — it measures distance by sending ultrasonic pulses and timing their echo.

How It Works

Send a 10µs HIGH pulse to TRIG. The sensor emits a burst of 40kHz sound. The ECHO pin goes HIGH for as long as it takes the sound to travel to an object and return. Distance:

Formula
distance_cm = pulse_duration_µs × 0.034 / 2 // ÷2 because sound travels TO the object AND BACK

Wiring

HC-SR04 PinDescriptionConnect To
VCCPowerArduino 5V
GNDGroundArduino GND
TRIGTrigger pulse outputArduino Pin 11
ECHOEcho input (goes HIGH)Arduino Pin 12
Arduino C++
#define TRIG_PIN 11 #define ECHO_PIN 12 void setup() { pinMode(TRIG_PIN, OUTPUT); pinMode(ECHO_PIN, INPUT); Serial.begin(9600); } float getDistance() { digitalWrite(TRIG_PIN, LOW); delayMicroseconds(2); digitalWrite(TRIG_PIN, HIGH); delayMicroseconds(10); digitalWrite(TRIG_PIN, LOW); long dur = pulseIn(ECHO_PIN, HIGH, 30000); // 30ms timeout return dur ? dur * 0.034 / 2.0 : 999; } void loop() { Serial.print("Distance: "); Serial.print(getDistance()); Serial.println(" cm"); delay(200); }
💡 Always Use the Timeout
The 30,000µs timeout in pulseIn() prevents your code from freezing forever if no echo returns. Without it, pointing the sensor into open space will hang your program.

IR Sensor

IR sensors detect surfaces and edges — perfect for line following and cliff detection.

The module emits infrared light. A light/white surface reflects it → output LOW. A dark surface or gap absorbs it → output HIGH.

IR Module PinDescriptionConnect To
VCCPowerArduino 5V
GNDGroundArduino GND
OUTDigital outputArduino Pin 7
Arduino C++
#define IR_PIN 7 void setup() { pinMode(IR_PIN, INPUT); Serial.begin(9600); } void loop() { if (digitalRead(IR_PIN) == LOW) Serial.println("Surface detected!"); delay(100); }
ℹ️ Adjustable Sensitivity
Most IR modules have a small blue potentiometer. Turn it with a screwdriver to adjust detection range (typically 0.5–3 cm).

Reading & Interpreting Sensor Data

Raw sensor data is noisy. Averaging and thresholding make your robot reliable.

Arduino C++ — Averaging + Two-Level Threshold
// Average 5 readings for stability float getStableDistance() { float sum = 0; for (int i = 0; i < 5; i++) { sum += getDistance(); delay(10); } return sum / 5; } const float DANGER = 15.0; // cm — stop and avoid const float CAUTION = 25.0; // cm — slow down void loop() { float d = getStableDistance(); if (d < DANGER) handleObstacle(); else if (d < CAUTION) slowDown(); else driveForward(200); }
