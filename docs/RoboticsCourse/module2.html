Voltage, Current & Resistance

Electronics is just the controlled flow of electrons. Master three core concepts and everything else follows.

The Water Analogy

The easiest way to understand electricity is to think about water in a pipe:

Electrical ConceptSymbol / UnitWater Analogy
Voltage (V)V / VoltsWater pressure pushing the flow
Current (I)A / AmperesFlow rate — litres per second
Resistance (R)Ω / OhmsPipe narrowness — restricts flow

Ohm's Law

These three always relate through one simple formula:

Formula
// Ohm's Law V = I × R // Rearranged forms: I = V ÷ R // Current = Voltage ÷ Resistance R = V ÷ I // Resistance = Voltage ÷ Current
ℹ️ Practical Example
An LED needs 2V at 20mA. Powered from Arduino's 5V pin:

Voltage across resistor = 5 − 2 = 3V
R = 3 ÷ 0.02 = 150Ω → use a 220Ω resistor to be safe.

Breadboard & Wiring

A breadboard lets you build circuits without soldering — perfect for prototyping.

How a Breadboard Works

  • Power rails (long strips marked + and −) — all holes in one strip are connected. Use for 5V and GND.
  • Terminal rows (short rows of 5 holes in the middle) — all 5 holes in a row are connected. The two halves across the centre gap are not connected.
💡 Colour Convention
Always use red wires for positive (5V/VCC) and black for ground (GND). This habit prevents many accidental short circuits.

Good Wiring Habits

  1. Keep wires short and tidy — long wires create mess and errors.
  2. Always connect GND first when adding a new component.
  3. Double-check polarity (+/−) before powering on.
  4. Never connect two power sources together unless you know what you're doing.
⚠️ Short Circuits
A short circuit connects + directly to − with no resistance. This causes extreme current flow and can destroy components or start a fire. Always check wiring before powering on.

LEDs, Resistors & Buttons

Your first hands-on circuits — the building blocks you'll use throughout the course.

The LED Circuit

An LED (Light Emitting Diode) must always be used with a current-limiting resistor, or it burns out instantly.

Circuit Diagram
Arduino Pin 13 ──── [220Ω] ──── LED (+) ──── LED (−) ──── GND

Reading a Button with INPUT_PULLUP

Using Arduino's built-in pull-up resistor means no external resistor is needed. Note: with INPUT_PULLUP, the logic is inverted — pressed = LOW.

Arduino C++
#define BTN_PIN 2 void setup() { pinMode(BTN_PIN, INPUT_PULLUP); // No external resistor needed Serial.begin(9600); } void loop() { if (digitalRead(BTN_PIN) == LOW) { // Pressed = LOW Serial.println("Button pressed!"); } delay(50); }
