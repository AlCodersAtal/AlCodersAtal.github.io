What is Flexbox?

Flexbox is a CSS layout system that makes aligning and distributing elements effortless.

Enabling Flexbox

Add display: flex to a container and all its direct children become flex items:

Flexbox
.container { display: flex; } /* Children automatically sit side by side */

Justify & Align

justify-content and align-items are the two most powerful flex properties — they control spacing and alignment.

justify-content — Horizontal Spacing

justify-content
.container { display: flex; justify-content: flex-start; /* default — left */ justify-content: flex-end; /* right */ justify-content: center; /* center */ justify-content: space-between;/* max space between */ justify-content: space-around; /* equal space around */ }

align-items — Vertical Alignment

align-items
.container { display: flex; height: 200px; align-items: flex-start; /* top */ align-items: center; /* middle */ align-items: flex-end; /* bottom */ align-items: stretch; /* fill height (default) */ }

Flex Direction & Wrap

Control whether items flow in a row or column, and what happens when they run out of space.

flex-direction

Direction
.container { flex-direction: row; /* default — left to right */ flex-direction: row-reverse; /* right to left */ flex-direction: column; /* top to bottom */ flex-direction: column-reverse; /* bottom to top */ } /* flex-wrap */ .container { flex-wrap: nowrap; /* default — all in one line */ flex-wrap: wrap; /* wrap to next line when needed */ } /* gap between items */ .container { gap: 16px; }

Building a Navbar with Flexbox

Navigation bars are one of the most common UI components. Flexbox makes them trivial to build.

A Classic Navbar Pattern

Navbar
<style> nav { display: flex; justify-content: space-between; align-items: center; background: #1a1a2e; padding: 16px 32px; } .logo { color: #1a5cff; font-weight: bold; font-size: 1.2rem; } .links { display: flex; gap: 24px; list-style: none; } .links a { color: #ccc; text-decoration: none; } </style> <nav> <span class="logo">AlCoders</span> <ul class="links"> <li><a href="#">Home</a></li> <li><a href="#">Courses</a></li> </ul> </nav>
