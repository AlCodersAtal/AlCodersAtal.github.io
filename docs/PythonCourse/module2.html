Variables & Assignment

Variables are containers that store data in your program. Every piece of information in Python — a name, a score, a temperature — lives in a variable.

What is a Variable?

Think of a variable as a labelled box. You put a value inside the box, and you can access or change that value later using the label.

Python
name = "Alice" # Variable 'name' holds the string "Alice" age = 16 # Variable 'age' holds the integer 16 height = 5.7 # Variable 'height' holds a decimal number is_student = True # Variable holds a boolean print(name) # Alice print(age) # 16

The = sign is called the assignment operator. It assigns the value on the right to the variable on the left. It does NOT mean "equal to" (that's ==).

Variable Naming Rules

RuleExample
Start with a letter or underscore✅ name, _age
Can contain letters, numbers, underscores✅ score1, my_value
Cannot start with a number❌ 1score
No spaces allowed❌ my score
Case sensitiveName ≠ name
No Python keywords❌ if, for, print

Naming Convention: snake_case

Python convention (PEP 8) is to use snake_case — all lowercase, words separated by underscores:

Python
# Good variable names student_name = "Bob" total_score = 95 is_passing = True max_attempts = 3 # Bad variable names (avoid) x = "Bob" # Too vague StudentName = "Bob" # CamelCase (used for classes, not variables)

Multiple Assignment

Python
# Assign same value to multiple variables x = y = z = 0 # Assign multiple variables in one line a, b, c = 1, 2, 3 print(a, b, c) # 1 2 3 # Swap two variables elegantly x, y = 10, 20 x, y = y, x # Now x=20, y=10

Numbers: int & float

Python has two main numeric types: integers (whole numbers) and floats (decimals). Understanding them is fundamental to all calculations.

Integers (int)

Integers are whole numbers — positive, negative, or zero. No decimal point.

Python
age = 25 temperature = -10 population = 1_000_000 # Underscores allowed for readability zero = 0 print(type(age)) # <class 'int'>

Python integers have unlimited precision — they can be as large as your computer's memory allows.

Floats (float)

Floats are numbers with a decimal point. They're used for fractional values.

Python
pi = 3.14159 price = 19.99 gravity = 9.81 very_small = 1.5e-4 # Scientific notation: 0.00015 very_large = 2.5e8 # 250,000,000 print(type(pi)) # <class 'float'>
⚠️ Float Precision Issues

Floats can have tiny rounding errors due to how computers store binary numbers. For example, 0.1 + 0.2 gives 0.30000000000000004. For financial calculations, use the decimal module instead.

Arithmetic with Numbers

Python
print(10 + 3) # 13 — addition print(10 - 3) # 7 — subtraction print(10 * 3) # 30 — multiplication print(10 / 3) # 3.3333... — division (always float) print(10 // 3) # 3 — floor division (integer result) print(10 % 3) # 1 — modulo (remainder) print(10 ** 3) # 1000 — exponentiation (10 to the power of 3)

Useful Math Functions

Python
print(abs(-15)) # 15 — absolute value print(round(3.7)) # 4 — round to nearest integer print(round(3.14159, 2)) # 3.14 — round to 2 decimal places print(max(3, 7, 1)) # 7 — largest value print(min(3, 7, 1)) # 1 — smallest value print(pow(2, 8)) # 256 — same as 2**8 import math print(math.sqrt(16)) # 4.0 — square root print(math.floor(3.9)) # 3 — round down print(math.ceil(3.1)) # 4 — round up

Strings

Strings are sequences of characters — text. They're one of the most-used data types in Python. Let's learn how to create and work with them.

Creating Strings

Python
name = "Alice" # Double quotes city = 'Mumbai' # Single quotes — both work quote = "It's a great day" # Single quote inside double quotes bio = """This is a multiline string spanning three lines.""" # Triple quotes for multiline

String Indexing & Length

Every character in a string has an index (position), starting from 0.

Python
word = "Python" # P y t h o n # 0 1 2 3 4 5 (positive index) # -6 -5 -4 -3 -2 -1 (negative index) print(word[0]) # P print(word[2]) # t print(word[-1]) # n (last character) print(len(word)) # 6 (number of characters)

Slicing

Python
text = "Hello, World!" print(text[0:5]) # Hello (index 0 up to but not including 5) print(text[7:]) # World! (from index 7 to end) print(text[:5]) # Hello (from start to index 5) print(text[::-1]) # !dlroW ,olleH (reversed!)

String Concatenation & Repetition

Python
first = "Hello" second = "World" print(first + " " + second) # Hello World print("Ha" * 3) # HaHaHa # Augmented assignment message = "Hello" message += " World" print(message) # Hello World

Escape Characters

Booleans & None

Booleans represent truth values — True or False. They're the foundation of all decision-making in programs. None represents the absence of a value.

Boolean Values

Python has exactly two boolean values: True and False (capital T and F — they're case-sensitive).

Python
is_raining = True is_sunny = False passed_exam = True print(type(True)) # <class 'bool'> # Comparisons produce booleans print(5 > 3) # True print(5 == 5) # True print(5 != 3) # True print(5 < 3) # False

Truthy and Falsy Values

In Python, any value can be evaluated as a boolean. Some values are "falsy" (behave like False), everything else is "truthy".

Falsy Values in Python

False, 0, 0.0, "" (empty string), [] (empty list), {} (empty dict), () (empty tuple), None

Python
print(bool(1)) # True print(bool(0)) # False print(bool("")) # False (empty string) print(bool("hi")) # True (non-empty string) print(bool([])) # False (empty list)

None — The Absence of Value

None is Python's way of saying "nothing" or "no value". It's similar to null in other languages.

Python
result = None print(result) # None print(type(result)) # <class 'NoneType'> print(result is None) # True — use 'is' not '==' for None # Functions that don't return anything return None implicitly def greet(): print("Hello") x = greet() print(x) # None

Type Conversion

Python lets you convert values from one data type to another. This is essential when working with user input (always a string) or mixing different numeric types.

type() — Checking the Type

Python
print(type(42)) # <class 'int'> print(type(3.14)) # <class 'float'> print(type("hello")) # <class 'str'> print(type(True)) # <class 'bool'> print(type(None)) # <class 'NoneType'>

Explicit Type Conversion

Python
# int() — convert to integer print(int("42")) # 42 print(int(3.9)) # 3 (truncates, doesn't round!) print(int(True)) # 1 print(int(False)) # 0 # float() — convert to float print(float("3.14")) # 3.14 print(float(7)) # 7.0 # str() — convert to string print(str(100)) # "100" print(str(3.14)) # "3.14" print(str(True)) # "True" # bool() — convert to boolean print(bool(0)) # False print(bool("hello")) # True
❌ Watch Out for ValueError

You can't convert just any string to int or float. int("hello") raises a ValueError. Only strings that contain valid numbers can be converted.

Implicit Conversion

Python automatically converts types in some situations:

Python
result = 5 + 2.5 # int + float = float (7.5) result = 10 / 4 # Division always returns float (2.5) result = True + 1 # True treated as 1, result = 2

Input from the User

Programs become interactive when they can receive input from the user. Python's input() function makes this easy.

The input() Function

Python
name = input("Enter your name: ") print("Hello, " + name + "!")
⚠️ input() always returns a string

Even if the user types a number, input() returns it as a string. You must convert it to int or float if you want to do math with it.

Getting Numeric Input

Python
age_str = input("Enter your age: ") age = int(age_str) print("In 10 years you will be", age + 10) # Shorter version — convert on same line height = float(input("Enter your height in metres: "))

Python
