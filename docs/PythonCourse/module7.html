String Methods

Strings have a rich set of built-in methods for transforming, searching, splitting, and validating text.

Case & Whitespace

Python
s = " Hello, World! " print(s.upper()) # " HELLO, WORLD! " print(s.lower()) # " hello, world! " print(s.title()) # " Hello, World! " print(s.strip()) # "Hello, World!" — remove leading/trailing spaces print(s.lstrip()) # remove left spaces only print(s.rstrip()) # remove right spaces only

Searching & Replacing

Python
text = "The quick brown fox jumps over the lazy dog" print(text.find("fox")) # 16 — index of first match, -1 if not found print(text.index("fox")) # 16 — like find() but raises ValueError if not found print(text.count("the")) # 1 (case-sensitive) print(text.replace("fox", "cat")) # "...brown cat jumps..." print(text.startswith("The")) # True print(text.endswith("dog")) # True

Splitting & Joining

Python
# split() — string to list csv_line = "Alice,25,Kochi,Engineer" parts = csv_line.split(",") print(parts) # ['Alice', '25', 'Kochi', 'Engineer'] sentence = "hello world python" words = sentence.split() # split on whitespace # join() — list to string items = ["one", "two", "three"] print(", ".join(items)) # "one, two, three" print(" | ".join(items)) # "one | two | three"

Validation Methods

Python
print("abc123".isalnum()) # True — letters and digits only print("hello".isalpha()) # True — letters only print("12345".isdigit()) # True — digits only print("hello".islower()) # True print("HELLO".isupper()) # True print(" ".isspace()) # True — whitespace only

String Formatting (f-strings)

f-strings (Python 3.6+) are the modern, powerful way to embed values and expressions inside strings.

f-string Basics

Python
name = "Alice" age = 25 score = 95.6789 # Basic embedding print(f"Name: {name}, Age: {age}") # Expressions inside {} print(f"Next year: {age + 1}") print(f"Name upper: {name.upper()}") print(f"Even? {age % 2 == 0}") # Format specifiers print(f"Score: {score:.2f}") # 95.68 — 2 decimal places print(f"Score: {score:10.2f}") # " 95.68" — 10 wide, right-aligned print(f"Percent: {0.856:.1%}") # 85.6% print(f"Comma: {1000000:,}") # 1,000,000 print(f"Hex: {255:x}") # ff print(f"Binary: {10:b}") # 1010 # Debug mode (Python 3.8+) print(f"{name=}") # name='Alice' — shows name and value

Alignment & Padding

Python
items = [("Apple", 1.5), ("Banana", 0.75), ("Cherry", 3.0)] for name, price in items: print(f"{name:10} £{price:>6.2f}") # Apple £ 1.50 # Banana £ 0.75 # Cherry £ 3.00

Regular Expressions

Regular expressions (regex) are a mini-language for searching and matching patterns in text. They're incredibly powerful for data extraction and validation.

The re Module

Python
import re text = "My phone is 9876543210 and email is alice@example.com" # re.search() — find first match anywhere in string m = re.search(r"\d{10}", text) if m: print(m.group()) # 9876543210 # re.findall() — find all matches emails = re.findall(r"[\w.+-]+@[\w-]+\.\w+", text) print(emails) # ['alice@example.com'] # re.sub() — replace pattern clean = re.sub(r"\d+", "[NUM]", text) print(clean) # "My phone is [NUM] and email is..." # re.match() — match at start of string only m = re.match(r"My", text) print(bool(m)) # True

Common Regex Patterns

PatternMatches
\dAny digit 0-9
\wWord char (letter, digit, _)
\sWhitespace (space, tab, newline)
.Any character except newline
^Start of string
$End of string
+1 or more
*0 or more
?0 or 1 (optional)
{n}Exactly n times

String Projects

Put your string skills to work with these practical mini-projects.

Project 1: Word Counter

Python
def word_count(text): """Count frequency of each word in text.""" words = text.lower().split() counts = {} for word in words: word = word.strip(".,!?;:") # remove punctuation counts[word] = counts.get(word, 0) + 1 return counts text = "the cat sat on the mat. the cat is fat." result = word_count(text) for word, count in sorted(result.items(), key=lambda x: -x[1]): print(f"{word:10} {count}")

Project 2: Palindrome Checker

Python
def is_palindrome(text): """Check if text is a palindrome (ignoring case/spaces).""" import re clean = re.sub(r"[^a-z0-9]", "", text.lower()) return clean == clean[::-1] print(is_palindrome("racecar")) # True print(is_palindrome("A man a plan a canal Panama")) # True print(is_palindrome("hello")) # False
