Module 4 · Lesson 1

if / elif / else

Conditions let your program make decisions — run different code depending on whether something is true or false. This is the heart of every program's logic.

The if Statement

Python
# Basic if temperature = 35 if temperature > 30: print("It's hot outside!") print("Drink water.") # Both indented lines run if condition is True

if / else

Python
score = 75 if score >= 50: print("You passed!") else: print("You failed.")

if / elif / else — Multiple Conditions

Python
score = 85 if score >= 90: print("Grade: A") elif score >= 80: print("Grade: B") elif score >= 70: print("Grade: C") elif score >= 60: print("Grade: D") else: print("Grade: F") # Output: Grade: B

Ternary / Conditional Expression

A compact one-line if-else:

Python
age = 20 status = "adult" if age >= 18 else "minor" print(status) # adult
Module 4 · Lesson 2

Nested Conditions

Conditions can be placed inside other conditions, creating layers of decision-making. Use this carefully — deep nesting makes code hard to read.

Nested if Statements

Python
age = 22 has_ticket = True if age >= 18: if has_ticket: print("Welcome! Enjoy the show.") else: print("You need a ticket.") else: print("Sorry, adults only.")
💡 Prefer 'and' over deep nesting

The same logic can often be written more clearly using and: if age >= 18 and has_ticket:. Only nest when the conditions are truly separate decisions.

match / case (Python 3.10+)

Python 3.10 introduced match statements — similar to switch in other languages:

Python 3.10+
command = "quit" match command: case "start": print("Starting...") case "stop" | "quit": # multiple options with | print("Stopping...") case _: # default (like else) print("Unknown command")
Module 4 · Lesson 3

for Loops

Loops let you repeat code. The for loop iterates over a sequence — a list, string, range, or any iterable — and runs the body once for each item.

Iterating over a List

Python
fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(fruit) # apple # banana # cherry

range() — Counting Loops

Python
# range(stop) — 0 to stop-1 for i in range(5): print(i) # 0, 1, 2, 3, 4 # range(start, stop) for i in range(1, 6): print(i) # 1, 2, 3, 4, 5 # range(start, stop, step) for i in range(0, 20, 5): print(i) # 0, 5, 10, 15 # Count backwards for i in range(10, 0, -1): print(i) # 10, 9, 8... 1

Iterating with enumerate()

Get both the index and value at the same time:

Python
names = ["Alice", "Bob", "Charlie"] for i, name in enumerate(names): print(f"{i}: {name}") # 0: Alice # 1: Bob # 2: Charlie
Module 4 · Lesson 4

while Loops

The while loop keeps running as long as a condition is True. Use it when you don't know in advance how many times to repeat.

Basic while Loop

Python
count = 0 while count < 5: print(f"Count: {count}") count += 1 # IMPORTANT: always update the condition variable! # Count: 0, 1, 2, 3, 4
❌ Infinite Loops

If you forget to update the condition variable, the loop runs forever. Always make sure your while condition will eventually become False. Press Ctrl+C to stop an infinite loop in terminal.

while with User Input

Python
password = "" while password != "secret123": password = input("Enter password: ") print("Access granted!")

while / else

Python's while loop can have an else clause that runs when the condition becomes False (but NOT if the loop was exited with break):

Python
n = 3 while n > 0: print(n) n -= 1 else: print("Loop finished!")
Module 4 · Lesson 5

break, continue, pass

These three keywords give you fine-grained control over loop execution — skipping iterations, stopping early, or doing nothing.

break — Exit the Loop Early

Python
# Find first even number numbers = [1, 3, 7, 4, 9] for num in numbers: if num % 2 == 0: print(f"Found even: {num}") break # Stop immediately # Found even: 4

continue — Skip to Next Iteration

Python
# Print only odd numbers for i in range(1, 11): if i % 2 == 0: continue # Skip even numbers, go to next i print(i) # 1, 3, 5, 7, 9

pass — Do Nothing (Placeholder)

Python
# pass is useful when you need a block but haven't written code yet for i in range(5): pass # TODO: implement later if True: pass # Empty if block — valid Python class MyClass: pass # Empty class skeleton

Loop Patterns & Practice

Loops have many common patterns. Learning to recognise them makes you a much faster programmer.

Common Patterns

Python
# 1. Accumulator — sum up values total = 0 for n in [10, 20, 30, 40]: total += n print(total) # 100 # 2. Counter — count matching items words = ["cat", "dog", "cat", "fish", "cat"] count = 0 for w in words: if w == "cat": count += 1 print(count) # 3 # 3. Find maximum scores = [42, 87, 65, 91, 34] best = scores[0] for s in scores: if s > best: best = s print(best) # 91 # 4. Build a list with a loop squares = [] for i in range(1, 6): squares.append(i ** 2) print(squares) # [1, 4, 9, 16, 25] # 5. Multiplication table for i in range(1, 11): print(f"5 × {i} = {5 * i}")

Nested Loops

Python
# Star pattern using nested loops for row in range(1, 6): for col in range(row): print("*", end="") print() # newline after each row # * # ** # *** # **** # *****
