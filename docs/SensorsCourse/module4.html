LDR — How Light Sensing Works Analog

The LDR (Light Dependent Resistor) is one of the most fundamental sensor components. Understanding it teaches analog sensing, voltage dividers, and how resistance converts to voltage.

What is an LDR?

An LDR (also called a photoresistor or photocell) is a passive resistor whose resistance changes dramatically depending on light intensity. In darkness: very high resistance (1MΩ or more). In bright light: low resistance (hundreds of ohms).

Why? LDRs are made from cadmium sulphide (CdS). Photons (light particles) hitting the CdS crystal excite electrons into the conduction band, creating more charge carriers — which lowers resistance. This is the photoconductive effect.

The Voltage Divider Circuit

Arduino reads voltage, not resistance. So we use a voltage divider to convert the LDR's changing resistance into a changing voltage. Connect the LDR and a fixed 10kΩ resistor in series between 5V and GND. Read the voltage at the midpoint.

Formula: Vout = 5V × R_fixed / (R_LDR + R_fixed)

  • Bright light → R_LDR small → Vout small → low analog reading
  • Darkness → R_LDR large → Vout approaches 5V → high analog reading

Arduino C++ — LDR Reading
const int LDR_PIN = A0; const int DARK_THRESHOLD = 500; void setup() { Serial.begin(9600); } void loop() { int ldrValue = analogRead(LDR_PIN); int lightPct = map(ldrValue, 0, 1023, 100, 0); // invert Serial.print("LDR: "); Serial.print(ldrValue); Serial.print(" | Light: "); Serial.print(lightPct); Serial.print("%"); Serial.println(ldrValue > DARK_THRESHOLD ? " 🌙 DARK" : " ☀ BRIGHT"); delay(500); }
✅ Calibration
Open Serial Monitor. Observe raw values in bright light (usually ~100–300) and in darkness (usually ~700–900). Set your threshold somewhere in between — typically 400–600 for indoor use.

LDR with Arduino — Full Examples

Let's build practical circuits with the LDR, including an automatic night light and using LDR modules with digital output.

Arduino C++ — Auto Night Light with PWM Dimming
const int LDR_PIN = A0; const int LED_PIN = 9; // PWM pin const int THRESHOLD = 500; void setup() { Serial.begin(9600); pinMode(LED_PIN, OUTPUT); } void loop() { int ldr = analogRead(LDR_PIN); if (ldr > THRESHOLD) { int brightness = map(ldr, THRESHOLD, 1023, 50, 255); analogWrite(LED_PIN, brightness); Serial.print("DARK — brightness: "); Serial.println(brightness); } else { analogWrite(LED_PIN, 0); Serial.println("Light OK — LED off"); } delay(200); }

Using an LDR Module (with DO output)

LDR modules (small PCB with LDR + LM393 comparator) output both AO (analog) and DO (digital threshold signal). Use the onboard potentiometer to set the light threshold.

Arduino C++ — LDR Module Digital Output
const int LDR_DO = 4; const int LDR_AO = A0; void setup() { Serial.begin(9600); pinMode(LDR_DO, INPUT); } void loop() { int dState = digitalRead(LDR_DO); int aVal = analogRead(LDR_AO); Serial.print(dState == LOW ? "Light" : "Dark"); Serial.print(" | Analog: "); Serial.println(aVal); delay(300); }

Photodiode Sensor Analog

Photodiodes are faster, more accurate, and more directional than LDRs. Used in line-following robots, fiber optic communication, and precision light measurement.

ComplexityVery simpleNeeds amplification
Common useLight/dark detectionLine following, remote RX

How a Photodiode Works

A photodiode is a p-n junction that generates current when photons strike it. It operates in reverse-biased mode. Under normal conditions, almost no current flows. When light hits the junction, photons create electron-hole pairs, allowing a small reverse current proportional to light intensity.

TCRT5000 — IR Photodiode Line Sensor

The TCRT5000 contains an IR LED (emitter) and photodiode (receiver) side by side. The IR LED emits light; the photodiode detects how much bounces back from a surface. Used for line-following robots and rotary encoders.

Arduino C++ — TCRT5000 Line Sensor
const int LINE_DO = 5; const int LINE_AO = A1; void setup() { Serial.begin(9600); pinMode(LINE_DO, INPUT); } void loop() { int digital = digitalRead(LINE_DO); int analog = analogRead(LINE_AO); Serial.print(digital == LOW ? "WHITE (clear)" : "BLACK (line)"); Serial.print(" | Analog: "); Serial.println(analog); delay(100); }

IR Sensor — Principle & Types Digital

Infrared sensors are everywhere — TV remotes, burglar alarms, and smartphone proximity sensors all use IR technology. Let's understand how they really work.

What is Infrared Light?

Infrared is electromagnetic radiation with wavelengths just beyond visible red light (700nm–1mm). We cannot see it, but it's emitted by everything warm including the human body. IR sensors exploit this in different ways.

Types of IR Sensors

  • Active IR sensors — have both an IR emitter (LED) and receiver (photodiode). Emit IR and detect how much bounces back. Used for object detection. Example: FC-51, TCRT5000.
  • Passive IR (PIR) sensors — detect IR emitted by warm objects (humans, animals). No emitter — receive only. Used for motion detection. Example: HC-SR501.
  • IR receiver modules — receive modulated IR signals from remote controls. Example: TSOP1738, VS1838B.
  • IR break-beam sensors — emitter and receiver on opposite sides of a gap. Detects when something breaks the beam.

FC-51 IR Obstacle Sensor Module

The most common IR sensor module for Arduino beginners. Contains an IR LED (constantly emitting 38kHz modulated IR), an IR receiver, an LM393 comparator, and a potentiometer to adjust detection distance (2–30cm). When an obstacle is in range, DO goes LOW. The onboard indicator LED lights up.

⚠️ IR Interference
IR sensors can be fooled by sunlight and incandescent bulbs, which emit strong IR radiation. If your sensor behaves erratically near windows, this is why.

IR Sensor with Arduino — Full Implementation

Let's build an obstacle detection system and decode remote control signals — one of the most satisfying beginner projects.

Arduino C++ — IR Obstacle Avoidance (FC-51)
const int IR_PIN = 4; const int BUZZ_PIN = 8; void setup() { Serial.begin(9600); pinMode(IR_PIN, INPUT); pinMode(BUZZ_PIN, OUTPUT); } void loop() { if (digitalRead(IR_PIN) == LOW) { // LOW = obstacle detected digitalWrite(BUZZ_PIN, HIGH); Serial.println("🚨 OBSTACLE DETECTED!"); } else { digitalWrite(BUZZ_PIN, LOW); Serial.println("✅ Path clear"); } delay(100); }

Decoding IR Remote Controls

Using a VS1838B IR receiver module and the IRremote library, you can read signals from any TV remote. Every button on a remote sends a unique hex code — you capture it and use it to trigger actions.

Arduino C++ — IR Remote Decoder
#include <IRremote.h> const int IR_RECV_PIN = 11; IRrecv irrecv(IR_RECV_PIN); decode_results results; void setup() { Serial.begin(9600); irrecv.enableIRIn(); Serial.println("IR Receiver ready — press a button"); } void loop() { if (irrecv.decode(&results)) { Serial.print("Code: 0x"); Serial.println(results.value, HEX); if (results.value == 0xFF6897) { // replace with your remote's code digitalWrite(13, HIGH); } irrecv.resume(); } delay(10); }
✅ Find Your Remote's Codes
Upload the code, open Serial Monitor, and press each button on your remote. Write down the hex code for each button. Then use those codes in if/switch statements to trigger actions.

Project: Automatic Night Lamp

Combine LDR light sensing with IR remote control for a smart lamp with auto and manual modes.

🔨 Build: Auto Night Lamp with IR Override

LED brightness increases automatically as the room gets darker. Manual override via IR remote lets you control brightness by hand.

  1. Wire LDR (A0), LED (pin 9 PWM), IR receiver (pin 11)
  2. In auto mode: LED brightness increases proportionally as room darkens
  3. Press remote button "1" to switch to manual mode (fixed brightness)
  4. Press "+" and "−" to adjust brightness manually in steps of 25
  5. Press "0" to return to auto mode
  6. Print current mode and brightness to Serial Monitor each loop
