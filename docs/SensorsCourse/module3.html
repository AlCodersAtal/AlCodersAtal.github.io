Push Button Sensor Digital

The push button is the simplest possible sensor. Learning it correctly teaches you principles that apply to every digital sensor.

What is a Push Button?

A push button (momentary tactile switch) creates or breaks an electrical connection when pressed. Most standard push buttons are normally open (NO) — the circuit is open (no connection) by default, and closes when pressed.

Internally it has two pairs of pins. Pressing the button bridges a metal contact between the two pairs.

Wiring with INPUT_PULLUP (Simplest Method)

Button PinConnect To
One legArduino Digital Pin 2
Other legGND

No external resistor needed — INPUT_PULLUP activates Arduino's internal 20kΩ pull-up resistor.

Arduino C++ — Button with INPUT_PULLUP
const int BUTTON_PIN = 2; const int LED_PIN = 13; void setup() { Serial.begin(9600); pinMode(BUTTON_PIN, INPUT_PULLUP); pinMode(LED_PIN, OUTPUT); } void loop() { int btnState = digitalRead(BUTTON_PIN); if (btnState == LOW) { // LOW = pressed with INPUT_PULLUP digitalWrite(LED_PIN, HIGH); Serial.println("Button PRESSED"); } else { digitalWrite(LED_PIN, LOW); } delay(50); }

Switch Debouncing

Mechanical buttons "bounce" rapidly for a few milliseconds when pressed, causing multiple false readings. Always debounce in real projects:

Arduino C++ — Debounced Button
const int BUTTON_PIN = 2; unsigned long lastDebounceTime = 0; const unsigned long DEBOUNCE_DELAY = 50; int lastButtonState = HIGH; int buttonState; int pressCount = 0; void setup() { Serial.begin(9600); pinMode(BUTTON_PIN, INPUT_PULLUP); } void loop() { int reading = digitalRead(BUTTON_PIN); if (reading != lastButtonState) lastDebounceTime = millis(); if ((millis() - lastDebounceTime) > DEBOUNCE_DELAY) { if (reading != buttonState) { buttonState = reading; if (buttonState == LOW) { pressCount++; Serial.println(pressCount); } } } lastButtonState = reading; }

Potentiometer Analog

The potentiometer is the most important analog sensor for beginners — it perfectly demonstrates how analog signals work and how Arduino reads them.

What is a Potentiometer?

A potentiometer (pot) is a variable resistor with a rotary knob. It has three pins: two ends of a resistive track and a wiper that slides along the track. As you turn the knob, the wiper moves and the voltage on the middle pin changes from 0V to 5V — a perfect analog signal.

Wiring

Potentiometer PinConnect To
Left pin5V
Middle pin (wiper)A0
Right pinGND
Arduino C++ — Potentiometer + LED Dimmer
const int POT_PIN = A0; const int LED_PIN = 9; // PWM pin void setup() { Serial.begin(9600); pinMode(LED_PIN, OUTPUT); } void loop() { int raw = analogRead(POT_PIN); // 0–1023 float voltage = (raw / 1023.0) * 5.0; // 0–5V int brightness = map(raw, 0, 1023, 0, 255); // map to PWM range analogWrite(LED_PIN, brightness); Serial.print("Raw: "); Serial.print(raw); Serial.print(" | Voltage: "); Serial.print(voltage); Serial.println("V"); delay(100); }
📌 The map() Function
map(value, fromLow, fromHigh, toLow, toHigh) linearly re-maps a number from one range to another. Above it converts 0–1023 (ADC range) to 0–255 (PWM range). You'll use this constantly with analog sensors.
Module 03 · Lesson 3

Tilt & Vibration Sensors Digital

Tilt and vibration sensors are simple but useful — perfect for burglar alarms, orientation detection, and shake-activated projects.

Tilt Sensor (SW-520D)

A tilt sensor contains a small metal ball inside a cylindrical tube with two electrodes. When upright, the ball bridges the electrodes and closes the circuit (LOW output). When tilted beyond a threshold angle, the ball rolls away and opens the circuit (HIGH output). It's a gravity-activated switch.

Vibration Sensor (SW-18010P)

Contains a spring-coiled wire inside a cylinder. When vibrated, the spring makes intermittent contact with the outer wall, producing rapid on/off pulses. Module versions include a comparator chip that outputs a clean digital HIGH when vibration exceeds a threshold — adjustable via an onboard potentiometer.

Wiring Tilt/Vibration Module

Module PinConnect To
VCC5V
GNDGND
DO (Digital Output)Digital Pin 3
AO (Analog Output)A0 (optional)
Arduino C++ — Tilt/Vibration Alarm
const int SENSOR_PIN = 3; const int BUZZER_PIN = 8; void setup() { Serial.begin(9600); pinMode(SENSOR_PIN, INPUT); pinMode(BUZZER_PIN, OUTPUT); } void loop() { if (digitalRead(SENSOR_PIN) == HIGH) { digitalWrite(BUZZER_PIN, HIGH); Serial.println("⚠ Tilt/Vibration Detected!"); delay(500); } else { digitalWrite(BUZZER_PIN, LOW); } delay(50); }

Sound / Microphone Sensor Analog + Digital

The KY-038 sound sensor module can detect claps, voices, and ambient noise — enabling clap-activated switches and sound-responsive projects.

How the Sound Sensor Works

The KY-038 module has an electret microphone that converts sound pressure waves into a tiny electrical signal. An onboard LM393 comparator amplifies this and compares it to a threshold set by an onboard potentiometer. If sound exceeds the threshold, the digital output (DO) goes HIGH. The analog output (AO) varies continuously with sound level.

Wiring

KY-038 PinConnect To
VCC5V
GNDGND
DODigital Pin 7
AOA0
Arduino C++ — Clap Switch
const int SOUND_DO = 7; const int SOUND_AO = A0; const int LED_PIN = 13; bool ledState = false; void setup() { Serial.begin(9600); pinMode(SOUND_DO, INPUT); pinMode(LED_PIN, OUTPUT); } void loop() { int soundLevel = analogRead(SOUND_AO); Serial.print("Sound level: "); Serial.println(soundLevel); if (digitalRead(SOUND_DO) == HIGH) { ledState = !ledState; digitalWrite(LED_PIN, ledState ? HIGH : LOW); Serial.println("👏 Clap detected!"); delay(300); // prevent double-trigger } delay(10); }
✅ Calibration Tip
Adjust the small blue potentiometer on the KY-038 to set sensitivity. The onboard red LED lights up when sound exceeds the threshold — use it for visual calibration without opening the Serial Monitor.
Module 03 · Project

Project: Interactive Sensor Dashboard

Combine the button, potentiometer, and sound sensor into a live Serial Monitor dashboard.

🔨 Build: Interactive Sensor Dashboard

Read all three Module 3 sensors simultaneously and display a formatted live dashboard in the Serial Monitor.

  1. Wire: button (pin 2), potentiometer (A0), sound sensor AO (A1), sound DO (pin 7)
  2. Every 200ms, read all sensors
  3. Print a formatted table: button state, pot percentage (0–100%), sound level (0–1023)
  4. When sound raw value exceeds 600, print "🔊 LOUD!" in the output
  5. When button is pressed, print "=== BUTTON PRESSED ===" in the output
Arduino C++ — Multi-Sensor Dashboard
const int BTN_PIN = 2; const int POT_PIN = A0; const int SOUND_AO = A1; const int SOUND_DO = 7; void setup() { Serial.begin(9600); pinMode(BTN_PIN, INPUT_PULLUP); pinMode(SOUND_DO, INPUT); Serial.println("=== Sensor Dashboard ==="); } void loop() { bool btnPressed = (digitalRead(BTN_PIN) == LOW); int potRaw = analogRead(POT_PIN); int potPct = map(potRaw, 0, 1023, 0, 100); int soundLevel = analogRead(SOUND_AO); bool soundLoud = (digitalRead(SOUND_DO) == HIGH); Serial.print("BTN: "); Serial.print(btnPressed ? "PRESSED" : "off "); Serial.print(" | POT: "); Serial.print(potPct); Serial.print("%"); Serial.print(" | SOUND: "); Serial.print(soundLevel); if (soundLoud) Serial.print(" 🔊 LOUD!"); Serial.println(); delay(200); }
