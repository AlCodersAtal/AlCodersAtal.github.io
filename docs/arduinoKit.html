1. Laser Security System

Project Overview:

The Laser Security System detects when an object interrupts a laser beam. When the laser is blocked, the LDR detects a drop in light intensity, triggering an alarm.

Components Required:

  • 1 x Arduino Uno
  • 1 x Laser module (5V)
  • 1 x LDR (Light Dependent Resistor)
  • 1 x 10kŒ© Resistor
  • 1 x Buzzer
  • Jumper wires
  • Breadboard

Physics Concept:

Light Detection: LDRs vary their resistance based on light intensity.
Security Systems: Used in real-world alarms.
Ohm‚Äôs Law: V = IR ‚Äì Resistance affects voltage that Arduino reads.

Circuit Connections:

  1. LDR: One end to 5V, other to A0 and GND through 10kŒ© resistor.
  2. Laser: Connect to digital pin 9 and GND.
  3. Buzzer: Connect to digital pin 8.

Arduino Code:

const int LDR = A0;
const int buzzer = 8;
const int laser = 9;

void setup() {
    pinMode(buzzer, OUTPUT);
    pinMode(laser, OUTPUT);
    pinMode(LDR, INPUT);
    Serial.begin(9600);
    digitalWrite(laser, HIGH); // Turn on laser
}

void loop() {
    int lightValue = analogRead(LDR);
    Serial.println(lightValue);
    
    if (lightValue < 300) {
        digitalWrite(buzzer, HIGH);
    } else {
        digitalWrite(buzzer, LOW);
    }
    delay(500);
}

How to Test:

  1. Upload the code to Arduino.
  2. Make sure laser is aligned with LDR.
  3. Block the laser beam.
  4. If buzzer sounds, it's working!

Possible Improvements:

  • Trigger relay-based alarms.
  • Add wireless alerts via Bluetooth/Wi-Fi.

2. RGB Color Mixer

Project Overview:

The RGB Color Mixer lets you control the brightness of Red, Green, and Blue LEDs using potentiometers.

Components Required:

  • 1 x Arduino Uno
  • 1 x RGB LED (Common Cathode)
  • 3 x Potentiometers (10kŒ©)
  • Jumper wires & Breadboard

Physics Concept:

Color Mixing: Combine RGB to create any color.
Additive Color Model: Used in screens.
PWM Control: Potentiometers adjust LED brightness via analog signals.

Circuit Connections:

  1. RGB LED:
    • Red ‚Üí D9
    • Green ‚Üí D10
    • Blue ‚Üí D11
    • Common cathode ‚Üí GND
  2. Potentiometers:
    • Middle pins ‚Üí A0, A1, A2
    • Sides ‚Üí 5V and GND

Arduino Code:

const int redPin = 9;
const int greenPin = 10;
const int bluePin = 11;
const int potRed = A0;
const int potGreen = A1;
const int potBlue = A2;

void setup() {
    pinMode(redPin, OUTPUT);
    pinMode(greenPin, OUTPUT);
    pinMode(bluePin, OUTPUT);
    Serial.begin(9600);
}

void loop() {
    int redVal = analogRead(potRed) / 4;
    int greenVal = analogRead(potGreen) / 4;
    int blueVal = analogRead(potBlue) / 4;
    
    analogWrite(redPin, redVal);
    analogWrite(greenPin, greenVal);
    analogWrite(bluePin, blueVal);
    
    Serial.print("R: "); Serial.print(redVal);
    Serial.print(" G: "); Serial.print(greenVal);
    Serial.print(" B: "); Serial.println(blueVal);
    delay(500);
}

How to Test:

  1. Upload code to Arduino.
  2. Turn potentiometers to adjust color.
  3. Observe color changes in RGB LED.

Possible Improvements:

  • Preset color buttons.
  • Bluetooth color control via mobile app.
