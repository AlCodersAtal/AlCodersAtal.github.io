How to make an LDR controlled LED

An LDR-controlled LED uses a Light Dependent Resistor to sense light levels and control an LED's state based on the surrounding brightness. The LDR forms a voltage divider with a resistor, and its output voltage changes with light intensity. This voltage is read by the Arduino through an analog pin. The Arduino compares the reading to a predefined threshold to decide whether to turn the LED on or off. If the environment is dark (high LDR resistance, higher voltage), the LED turns on; in bright conditions (low LDR resistance, lower voltage), the LED turns off. The circuit consists of an LDR connected to a 10kΩ resistor forming a voltage divider, with one end connected to 5V, the other to ground, and the midpoint connected to the Arduino’s analog pin. The LED is connected to a digital pin via a 220Ω resistor.

Here's the code:

You can copy it and paste it in the Arduino IDE.

        const int ldrPin = A0;       // LDR connected to analog pin A0
        const int ledPin = 13;       // LED connected to digital pin 13
        int threshold = 500;         // Threshold value to determine light/dark

        void setup() {
            pinMode(ledPin, OUTPUT);   // Set the LED pin as OUTPUT
            Serial.begin(9600);        // Start serial communication for debugging
        }

        void loop() {
            int ldrValue = analogRead(ldrPin); // Read the value from LDR
            Serial.println(ldrValue);         // Print the LDR value for debugging

            if (ldrValue < threshold) {       // If light is bright (low resistance)
                digitalWrite(ledPin, LOW);    // Turn off the LED
            } else {                          // If it is dark (high resistance)
                digitalWrite(ledPin, HIGH);   // Turn on the LED
            }

            delay(100);  // Small delay for stability
        }

Working:

The LDR-controlled LED circuit uses an LDR to sense light levels. The LDR is connected to an analog pin of the Arduino, which reads the voltage across the LDR. The Arduino compares this voltage to a predefined threshold value to determine whether the environment is bright or dark. If the light is bright (low resistance), the LED is turned off. If the light is dark (high resistance), the LED is turned on. This circuit can be used for applications like automatic night lights, streetlights, and more.

Applications:

  • Automatic Night Lights
  • Streetlights
  • Security Lighting
  • Energy-Efficient Lighting Systems

Conclusion:

The LDR-controlled LED circuit is a simple and effective way to control an LED based on light levels. This circuit can be used in various applications, including automatic night lights, streetlights, and security lighting.
