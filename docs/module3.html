Semantic Elements

Semantic tags tell the browser what role each part of your page plays — making your code smarter and more accessible.

What is Semantic HTML?

Non-semantic HTML uses generic containers like <div> everywhere. Semantic HTML uses tags that describe their purpose:

Semantic Layout
<header> <!-- Site header, logo, nav --> <nav> <!-- Navigation links --> <main> <!-- Main content of the page --> <section> <!-- A thematic group of content --> <article> <!-- A self-contained piece (blog post, card) --> <aside> <!-- Sidebar content --> <footer> <!-- Site footer -->
💡 Why Semantic HTML?
Search engines rank pages higher, screen readers work better, and your code is easier to read and maintain.

A Full Page Layout

Full Layout
<header> <h1>My Website</h1> <nav> <a href="index.html">Home</a> <a href="about.html">About</a> </nav> </header> <main> <section> <h2>Welcome</h2> <p>Hello, welcome to my site!</p> </section> </main> <footer> <p>&copy; 2025 My Website</p> </footer>

Divs & Spans

When no semantic tag fits, <div> and <span> are your generic containers for grouping and styling elements.

<div> — Block Container

A <div> groups block-level elements and takes up the full width. Used for layout sections.

div
<div class="card"> <h2>Card Title</h2> <p>Card content here.</p> </div>

<span> — Inline Container

A <span> wraps inline elements (inside text) without breaking onto a new line.

span
<p>The price is <span class="price">$29.99</span> per month.</p> <p>Status: <span class="status">Active</span></p>
💡 div vs span
Use <div> to group multiple elements in a block. Use <span> to style or target a specific word or phrase inside text.

Building Forms

Forms let users send you information — from login fields to contact messages. Learn to build them properly.

The <form> Tag

Basic Form
<form action="submit.php" method="post"> <label for="name">Your Name:</label> <input type="text" id="name" name="name" placeholder="Enter name"> <label for="email">Email:</label> <input type="email" id="email" name="email"> <label for="msg">Message:</label> <textarea id="msg" name="message" rows="4"></textarea> <button type="submit">Send</button> </form>
💡 Labels are important
Always pair each input with a <label> using matching for and id values. It improves accessibility and clicking the label focuses the input!
