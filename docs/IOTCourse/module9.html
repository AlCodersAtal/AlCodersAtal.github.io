Project: Cloud Temperature Monitor

Your first complete IoT project: a DHT11 sensor on an ESP32 that logs temperature and humidity to ThingSpeak every 30 seconds and displays a live chart in your browser.

What You'll Build

  • ESP32 reads temperature & humidity from DHT11 every 30 seconds
  • Data is sent to ThingSpeak via HTTP GET
  • ThingSpeak dashboard shows live updating line charts
  • Bonus: configure a ThingSpeak alert if temperature exceeds 35°C

Components Needed

ESP32
Main microcontroller with Wi-Fi
DHT11 Sensor
Temperature + humidity
10kΩ Resistor
Pull-up on DHT11 data pin
Breadboard + Wires
For the circuit

Complete Project Code

Arduino (ESP32)
#include <WiFi.h> #include <HTTPClient.h> #include <DHT.h> const char* ssid = "YourWiFi"; const char* password = "YourPass"; const char* apiKey = "YOUR_THINGSPEAK_WRITE_KEY"; #define DHTPIN 4 #define DHTTYPE DHT11 DHT dht(DHTPIN, DHTTYPE); unsigned long lastSend = 0; const long interval = 30000; // 30 seconds void setup() { Serial.begin(115200); dht.begin(); WiFi.begin(ssid, password); Serial.print("Connecting"); while (WiFi.status() != WL_CONNECTED) { delay(500); Serial.print("."); } Serial.println("\nConnected! IP: " + WiFi.localIP().toString()); } void sendToThingSpeak(float temp, float hum) { HTTPClient http; String url = String("https://api.thingspeak.com/update?api_key=") + apiKey + "&field1=" + temp + "&field2=" + hum; http.begin(url); int code = http.GET(); Serial.println("Response: " + String(code)); http.end(); } void loop() { unsigned long now = millis(); if (now - lastSend >= interval) { lastSend = now; float t = dht.readTemperature(); float h = dht.readHumidity(); if (!isnan(t) && !isnan(h)) { Serial.printf("T=%.1f°C H=%.1f%%\n", t, h); sendToThingSpeak(t, h); } else { Serial.println("DHT read failed"); } } }

Build Steps

  1. Wire DHT11 to ESP32 (VCC→3.3V, GND→GND, DATA→GPIO4 with 10kΩ pull-up to 3.3V)
  2. Create a ThingSpeak account and new Channel with two fields: Temperature and Humidity
  3. Copy your Write API Key from the API Keys tab
  4. Paste your Wi-Fi credentials and API key into the code
  5. Install the DHT sensor library by Adafruit in Library Manager
  6. Upload the code and open Serial Monitor at 115200 baud
  7. Open your ThingSpeak channel — watch the charts fill in!

Project: Smart RGB LED Controller

Control an RGB LED from anywhere in the world using Adafruit IO. A colour picker on the dashboard sends RGB values to your ESP32, which sets the LED colour in real time.

Wi-Fi microcontroller
Common Cathode RGB LED
3 colours in one package
3× 220Ω Resistors
Current limiting for each LED pin
Breadboard + Wires
For the circuit

RGB LED Wiring

RGB LED PinESP32 PinNote

G (Green)GPIO 26Via 220Ω resistor
B (Blue)GPIO 27Via 220Ω resistor
Common (–)GNDLongest pin

Complete Project Code

Arduino (ESP32)
#include <WiFi.h> #include <Adafruit_MQTT.h> #include <Adafruit_MQTT_Client.h> #define AIO_USERNAME "your_username" #define AIO_KEY "your_aio_key" #define PIN_R 25 #define PIN_G 26 #define PIN_B 27 WiFiClient client; Adafruit_MQTT_Client mqtt(&client, "io.adafruit.com", 1883, AIO_USERNAME, AIO_KEY); Adafruit_MQTT_Subscribe colorFeed(&mqtt, AIO_USERNAME "/feeds/color"); void setColor(int r, int g, int b) { analogWrite(PIN_R, r); analogWrite(PIN_G, g); analogWrite(PIN_B, b); } void setup() { Serial.begin(115200); pinMode(PIN_R, OUTPUT); pinMode(PIN_G, OUTPUT); pinMode(PIN_B, OUTPUT); WiFi.begin("SSID", "PASS"); while (WiFi.status() != WL_CONNECTED) delay(500); mqtt.subscribe(&colorFeed); mqtt.connect(); } void loop() { mqtt.processPackets(1000); if (!mqtt.connected()) mqtt.connect(); Adafruit_MQTT_Subscribe* sub = mqtt.readSubscription(100); if (sub == &colorFeed) { String hex = (char*)colorFeed.lastread; // Parse #RRGGBB hex colour from Adafruit IO colour picker long colour = strtol(hex.substring(1).c_str(), nullptr, 16); int r = (colour >> 16) & 0xFF; int g = (colour >> 8) & 0xFF; int b = colour & 0xFF; setColor(r, g, b); Serial.printf("Colour set: R=%d G=%d B=%d\n", r, g, b); } }

Build Steps

  1. Wire RGB LED to ESP32 as per the wiring table above
  2. Create an Adafruit IO feed named color
  3. Add a Color Picker widget to your dashboard, linked to the color feed
  4. Paste your AIO username and key into the code
  5. Install Adafruit MQTT Library from Library Manager
  6. Upload and open Serial Monitor — you should see "Connected"
  7. Click a colour in your Adafruit IO dashboard — LED changes instantly!

Project: Motion-Triggered Security Alert

A PIR motion sensor watches a room. When motion is detected, the ESP32 sends an IFTTT webhook that logs the event to Google Sheets and sends a Telegram message.

System Overview

  • PIR sensor triggers on movement → ESP32 detects HIGH signal
  • ESP32 calls IFTTT Webhooks URL with motion timestamp
  • IFTTT logs a row to Google Sheets (time, location, event)
  • IFTTT sends Telegram message: "Motion detected at Front Door — 14:32"
  • 30-second cooldown prevents repeated alerts

Wiring: PIR to ESP32

PIR PinESP32 Pin
VCC5V (Vin)
OUTGPIO 14
GNDGND

Complete Project Code

Arduino (ESP32)
#include <WiFi.h> #include <HTTPClient.h> const char* ssid = "YourWiFi"; const char* password = "YourPass"; const char* iftttKey = "YOUR_IFTTT_KEY"; const char* location = "Front Door"; #define PIR_PIN 14 #define COOLDOWN 30000UL unsigned long lastAlert = 0; void setup() { Serial.begin(115200); pinMode(PIR_PIN, INPUT); WiFi.begin(ssid, password); while (WiFi.status() != WL_CONNECTED) delay(500); Serial.println("Ready. Monitoring for motion..."); } void sendAlert() { HTTPClient http; String url = String("https://maker.ifttt.com/trigger/motion_detected/with/key/") + iftttKey; http.begin(url); http.addHeader("Content-Type", "application/json"); String body = String('{"value1":"') + location + '","value2":"' + millis()/1000 + '"}'; int code = http.POST(body); Serial.println("IFTTT response: " + String(code)); http.end(); } void loop() { if (digitalRead(PIR_PIN) == HIGH) { unsigned long now = millis(); if (now - lastAlert > COOLDOWN) { lastAlert = now; Serial.println("Motion detected! Sending alert..."); sendAlert(); } } delay(100); }

Build Steps

  1. Wire PIR to ESP32 as per wiring table. Power PIR from 5V/Vin, not 3.3V.
  2. Create IFTTT account. Go to Create → If Webhooks → Then Google Sheets
  3. Create a second Applet: If Webhooks → Then Telegram (install Telegram bot first)
  4. Find your IFTTT Webhooks key at ifttt.com/maker_webhooks/settings
  5. Paste Wi-Fi + IFTTT key into code. Set your location name.
  6. Upload and wave your hand in front of the PIR
  7. Check Google Sheets and Telegram — you should see the alert!

Project: Smart Plant Monitor

Never let your plant die again! This system monitors soil moisture and temperature, sends readings to Adafruit IO, and alerts you when the soil gets too dry.

Wiring

Soil sensorGNDGND
Soil sensorAOUTGPIO 34 (ADC)

Complete Project Code

Arduino (ESP32)
#include <WiFi.h> #include <Adafruit_MQTT.h> #include <Adafruit_MQTT_Client.h> #include <DHT.h> #define SOIL_PIN 34 #define DHT_PIN 4 #define DRY_VAL 2800 // calibrate: raw ADC value when dry #define WET_VAL 1200 // calibrate: raw ADC value in water DHT dht(DHT_PIN, DHT11); WiFiClient wc; Adafruit_MQTT_Client mqtt(&wc, "io.adafruit.com", 1883, "USER", "KEY"); Adafruit_MQTT_Publish soilFeed(&mqtt, "USER/feeds/soil"); Adafruit_MQTT_Publish tempFeed(&mqtt, "USER/feeds/temperature"); int soilPercent() { int raw = analogRead(SOIL_PIN); return constrain(map(raw, DRY_VAL, WET_VAL, 0, 100), 0, 100); } void setup() { Serial.begin(115200); dht.begin(); WiFi.begin("SSID", "PASS"); while (WiFi.status() != WL_CONNECTED) delay(500); mqtt.connect(); } void loop() { if (!mqtt.connected()) mqtt.connect(); int soil = soilPercent(); float temp = dht.readTemperature(); soilFeed.publish(soil); tempFeed.publish(temp); Serial.printf("Soil: %d%% | Temp: %.1f°C\n", soil, temp); delay(60000); // send every minute }
Calibrating the Soil Sensor
Read the ADC value with the sensor in dry air and note it as DRY_VAL. Then submerge the sensor tip in water and note the value as WET_VAL. These numbers vary between sensors — always calibrate yours.

Build Steps

  1. Wire soil sensor and DHT11 to ESP32 as per the table above
  2. Calibrate: upload a simple analogRead sketch, note dry vs wet values
  3. Create Adafruit IO feeds: soil and temperature
  4. Build a dashboard with two gauges and a line chart
  5. Set an Adafruit IO Trigger: if soil < 20, send email "Water your plant!"
  6. Upload final code, place sensor in plant soil
  7. Watch moisture drop over days as the plant drinks!

Project: Personal Weather Station

Build a hyperlocal weather station that measures temperature, humidity, and air pressure, displays live readings on an OLED screen, and publishes history to ThingSpeak.

Components Needed

ESP32
Wi-Fi microcontroller
BME280 Sensor
Temp + Humidity + Pressure (I²C)
SSD1306 OLED (128×64)
Local display (I²C)
Breadboard + Wires
Assembly

Wiring (Both Devices on I²C Bus)

DevicePinESP32
BME280 + OLEDVCC3.3V
BME280 + OLEDGNDGND
BME280 + OLEDSDAGPIO 21
BME280 + OLEDSCLGPIO 22

Both devices share the same I²C bus. BME280 address is usually 0x76 or 0x77; SSD1306 is 0x3C.

Complete Project Code

Arduino (ESP32)
#include <Wire.h> #include <Adafruit_BME280.h> #include <Adafruit_SSD1306.h> #include <WiFi.h> #include <HTTPClient.h> Adafruit_BME280 bme; Adafruit_SSD1306 display(128, 64, &Wire, -1); const char* apiKey = "YOUR_THINGSPEAK_KEY"; void setup() { Serial.begin(115200); bme.begin(0x76); display.begin(SSD1306_SWITCHCAPVCC, 0x3C); display.clearDisplay(); WiFi.begin("SSID", "PASS"); while (WiFi.status() != WL_CONNECTED) delay(500); } void updateOLED(float t, float h, float p) { display.clearDisplay(); display.setTextSize(1); display.setTextColor(WHITE); display.setCursor(0, 0); display.print("Temp: "); display.print(t); display.println(" C"); display.setCursor(0, 16); display.print("Humid: "); display.print(h); display.println(" %"); display.setCursor(0, 32); display.print("Press: "); display.print(p/100); display.println(" hPa"); display.display(); } void loop() { float t = bme.readTemperature(); float h = bme.readHumidity(); float p = bme.readPressure(); updateOLED(t, h, p); HTTPClient http; String url = String("https://api.thingspeak.com/update?api_key=") + apiKey + "&field1=" + t + "&field2=" + h + "&field3=" + (p/100); http.begin(url); http.GET(); http.end(); delay(30000); }

Build Steps

  1. Wire BME280 and SSD1306 to I²C bus (GPIO 21/22, shared)
  2. Install libraries: Adafruit BME280, Adafruit SSD1306, Adafruit GFX
  3. Create ThingSpeak channel with 3 fields: Temperature, Humidity, Pressure
  4. Paste Wi-Fi + API key into code
  5. Upload — the OLED should show readings within 5 seconds
  6. Visit ThingSpeak channel — all three charts will start filling in!
  7. Challenge: add a fourth field for a light sensor (LDR on GPIO 35)
