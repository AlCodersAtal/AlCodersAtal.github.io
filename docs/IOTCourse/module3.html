What are Sensors?

Sensors are the eyes, ears, nose, and skin of an IoT device. They convert physical phenomena into electrical signals that a microcontroller can read.

Sensors: The Physical-to-Digital Bridge

A sensor converts a physical quantity (temperature, light, pressure, movement) into an electrical signal (voltage or current). The microcontroller reads this electrical signal and converts it to a digital number your code can use.

Example: A temperature sensor produces a voltage that varies with temperature. At 25°C it might output 2.5V. The ESP32's ADC reads this as a number from 0–4095, and your code maps that to degrees.

Analog vs Digital Sensors

TypeOutputExamplesReading Pin
AnalogVariable voltage (0–5V)LDR, soil sensor, MQ gas sensorAnalog pin (A0, ADC)

Protocol-basedSerial data packetDHT11, BMP180, MPU6050Specific protocol (1-Wire, I2C)

Key Sensor Specifications

When choosing a sensor, check these specs:

  • Range: What values can it measure? (e.g., DHT11: 0–50°C)
  • Accuracy: How close to the real value? (DHT11: ±2°C)
  • Resolution: Smallest change it can detect (1°C for DHT11)
  • Response time: How fast does it react to changes?
  • Operating voltage: 3.3V or 5V? (critical — wrong voltage can destroy sensor)
  • Interface: Analog, digital, I2C, SPI, 1-Wire?

Reading an Analog Sensor

Arduino C
// Read an analog sensor on pin A0 (e.g., LDR or potentiometer) int sensorPin = A0; int sensorValue; void setup() { Serial.begin(9600); } void loop() { sensorValue = analogRead(sensorPin); // 0 to 1023 // Map to a meaningful range (e.g., 0–100%) int percentage = map(sensorValue, 0, 1023, 0, 100); Serial.print("Sensor: "); Serial.print(sensorValue); Serial.print(" | Percent: "); Serial.println(percentage); delay(500); }

Temperature & Humidity Sensors

Temperature and humidity sensors are the most common in IoT. They're used in weather stations, smart thermostats, greenhouses, server rooms, and more.

DHT11 — The Beginner's Choice

The DHT11 is the classic beginner temperature & humidity sensor. It's cheap, easy to use, and widely available in India (₹30–₹50).

📌 Pull-up Resistor
Add a 10kΩ resistor between VCC and DATA pin. Most DHT11 modules already include this.

DHT11 Code with ESP32

Arduino C (ESP32)
#include <DHT.h> #define DHT_PIN 4 #define DHT_TYPE DHT11 DHT dht(DHT_PIN, DHT_TYPE); void setup() { Serial.begin(115200); dht.begin(); Serial.println("DHT11 Ready!"); } void loop() { delay(2000); // DHT11 needs 2 sec between reads float temperature = dht.readTemperature(); float humidity = dht.readHumidity(); if (isnan(temperature) || isnan(humidity)) { Serial.println("Sensor read error!"); return; } Serial.print("Temperature: "); Serial.print(temperature); Serial.print("°C | Humidity: "); Serial.print(humidity); Serial.println("%"); }

Library needed: Install "DHT sensor library" by Adafruit in Arduino IDE's Library Manager.

Motion & Proximity Sensors

Motion and proximity sensors let your device know when something is near or moving. Essential for security systems, automatic lights, and presence detection.

PIR Sensor (Motion Detection)

PIR = Passive Infrared Sensor. It detects movement by sensing changes in infrared radiation (heat) emitted by humans and animals. It doesn't emit anything — it passively listens.

  • Detection range: up to 7 metres
  • Detection angle: ~110° cone
  • Output: Digital HIGH when motion detected
  • Operating voltage: 5V (most modules have a 3.3V-compatible output)
⚠️ Warm-up Time
PIR sensors take 30–60 seconds to warm up and stabilise after power-on. Don't check for motion immediately at startup.

PIR Code

Arduino C
int pirPin = 5; int ledPin = 13; int pirState = LOW; void setup() { pinMode(pirPin, INPUT); pinMode(ledPin, OUTPUT); Serial.begin(9600); Serial.println("Warming up PIR sensor..."); delay(30000); // 30 sec warm-up Serial.println("PIR ready!"); } void loop() { int val = digitalRead(pirPin); if (val == HIGH) { digitalWrite(ledPin, HIGH); if (pirState == LOW) { Serial.println("🚨 Motion detected!"); pirState = HIGH; } } else { digitalWrite(ledPin, LOW); if (pirState == HIGH) { Serial.println("No motion."); pirState = LOW; } } }

Ultrasonic Sensor (HC-SR04) — Distance Detection

The HC-SR04 measures distance by sending ultrasonic pulses and timing the echo. Range: 2cm–400cm, accuracy ±3mm.

Arduino C
#define TRIG_PIN 9 #define ECHO_PIN 10 void setup() { Serial.begin(9600); pinMode(TRIG_PIN, OUTPUT); pinMode(ECHO_PIN, INPUT); } long getDistance() { digitalWrite(TRIG_PIN, LOW); delayMicroseconds(2); digitalWrite(TRIG_PIN, HIGH); delayMicroseconds(10); digitalWrite(TRIG_PIN, LOW); long duration = pulseIn(ECHO_PIN, HIGH); return duration * 0.034 / 2; // cm } void loop() { long distance = getDistance(); Serial.print("Distance: "); Serial.print(distance); Serial.println(" cm"); delay(500); }

Light, Sound & Gas Sensors

Light, sound, and gas sensors open up a new world of environmental sensing — perfect for smart city, safety, and home automation applications.

LDR — Light Dependent Resistor

An LDR changes resistance based on light intensity. In bright light, resistance drops. In darkness, it rises significantly. Connect it with a 10kΩ resistor as a voltage divider to the analog input.

Arduino C
int ldrPin = A0; void loop() { int light = analogRead(ldrPin); if (light < 300) { Serial.println("It's dark — turn on the light!"); } else { Serial.println("Enough light."); } delay(1000); }

Sound Sensor (KY-038)

The KY-038 microphone module has both analog (sound level) and digital (threshold trigger) outputs. Use digital mode for clap detection, analog mode for measuring noise level.

🎙️ Use Case
Sound sensors are used in clap-activated switches, noise monitoring, and voice-activated triggers (not voice recognition — just sound presence detection).

MQ Gas Sensors

The MQ series are affordable gas sensors, each calibrated for different gases:

🔥

MQ-2

LPG, smoke, alcohol, propane

💨

MQ-3

Alcohol (breathalyser)

☠️

MQ-7

Carbon monoxide (CO)

🌫️

MQ-135

Air quality (CO2, NH3, NOx)

⚠️ Warm-up Required
MQ sensors need 20–60 seconds of warm-up after power-on before readings are accurate. They also draw 150–200mA — not suitable for battery power alone.
Arduino C
int mq2Pin = A0; int threshold = 400; // adjust for your environment void loop() { int gasLevel = analogRead(mq2Pin); Serial.print("Gas level: "); Serial.println(gasLevel); if (gasLevel > threshold) { Serial.println("⚠️ Gas detected! Alert!"); // Trigger buzzer or send cloud alert } delay(1000); }

Actuators: Motors, Relays & LEDs

Sensors read the world; actuators change it. Actuators are the output side of IoT — they turn data into physical action.

Types of Actuators

💡

LED

Light indicator, status display, controllable lighting

🔔

Buzzer

Audible alerts, alarms, simple tones

⚡

Relay

Switch high-voltage loads (lights, fans, appliances)

🔄

DC Motor

Rotation, wheels, fans, pumps

🎯

Servo Motor

Precise angle control: doors, valves, robot joints

💧

Solenoid

Electronic valve — open/close water/gas flow

Relay Module — Controlling Mains Power

A relay is an electrically operated switch. Your ESP32 (3.3V/5V) can control a relay, which then switches 230V AC mains safely. This is how smart plugs and smart switches work.

⚠️ DANGER — High Voltage
The AC mains side of a relay carries 230V which is lethal. Never touch the AC wiring when powered. If you're not comfortable with mains wiring, use only low-voltage DC loads for learning.
Arduino C
int relayPin = 8; void setup() { pinMode(relayPin, OUTPUT); digitalWrite(relayPin, HIGH); // Most relay modules: HIGH = OFF } void loop() { digitalWrite(relayPin, LOW); // Turn ON (activate relay) delay(5000); // On for 5 seconds digitalWrite(relayPin, HIGH); // Turn OFF delay(5000); // Off for 5 seconds }

Servo Motor — Precise Positioning

Arduino C
#include <Servo.h> Servo myServo; void setup() { myServo.attach(9); // Servo signal wire to pin 9 } void loop() { myServo.write(0); // Rotate to 0° delay(1000); myServo.write(90); // Rotate to 90° delay(1000); myServo.write(180); // Rotate to 180° delay(1000); }
What is the main purpose of a relay in an IoT system?
Measure current
Switch high-voltage loads using a low-voltage signal
Amplify sensor signals
Store data temporarily
✅ Exactly right! A relay lets your 3.3V ESP32 safely control 230V appliances.
❌ Not quite. A relay is a switch — it lets a low-voltage MCU control high-voltage loads safely.
