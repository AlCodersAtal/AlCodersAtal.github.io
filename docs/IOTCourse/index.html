FAQ

Common Questions

Do I need any prior experience? ▾
No experience needed! The course starts from the very basics. If you've done a little Arduino before, that helps but is not required.
What hardware do I need? ▾
For most lessons an Arduino Uno or ESP32 is enough. The projects list specific components. An ESP32 (≈₹350) covers most of the course.
Is the cloud part free? ▾
Yes! We use free tiers of ThingSpeak, Adafruit IO and Blynk — no credit card required for learning.
What programming language is used? ▾
Primarily Arduino C/C++, which is beginner-friendly. We also introduce MicroPython on ESP32 in Module 5.
Can I do this from home? ▾
Absolutely. All you need is a computer, an internet connection, and a basic hardware kit you can buy online.
