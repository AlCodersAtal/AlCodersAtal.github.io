Getting Started with Arduino

Welcome to Arduino! In this module you'll learn exactly what Arduino is, install all the tools you need, understand the board layout, and write your very first program.

🤖 Lesson 1: What is Arduino?

Arduino is an open-source electronics platform that combines a physical programmable circuit board (the microcontroller) with software (the IDE) that runs on your computer. You write code on your computer, upload it to the Arduino board, and the board runs your program — controlling lights, reading sensors, driving motors, and more.

Think of it as a tiny, programmable brain that you can connect to the real world.

💡
Key Idea Arduino is NOT just hardware, and NOT just software — it's both together. The board is the body, and your code is the brain.

Why Arduino?

  • ✅ Beginner friendly — simplified C++ language, huge community
  • ✅ Affordable — official boards from ₹500–₹2000, cheap clones even less
  • ✅ Huge ecosystem — thousands of libraries, shields, and sensor modules
  • ✅ Used professionally — rapid prototyping in startups, universities, and research labs
  • ✅ Open source — everything is free to learn, modify, and build upon

Popular Arduino Boards

✅
Recommendation For this course, use the Arduino Uno R3. It's the most widely documented board, and every tutorial in this course is built around it. You can use a clone (CH340 chip) to save money — it works identically.

What Can You Build?

Arduino is used for an incredible range of real-world projects:

  • 🌡️ Weather stations that post data online
  • 🤖 Robots that avoid obstacles or follow lines
  • 🏠 Smart home systems (lights, locks, fans)
  • 🎮 Custom game controllers
  • 💊 Automated medication dispensers
  • 🌱 Automated plant watering systems
  • 🔒 RFID-based door locks

🖥️ Lesson 2: Installing the Arduino IDE

The Arduino IDE (Integrated Development Environment) is the free software where you write, edit, and upload your code to the Arduino board. Let's install it step by step.

Download the IDE

1
Go to the official website

Open your browser and visit arduino.cc/en/software. Always download from the official site!

2
Choose your version

Download Arduino IDE 2.x (latest) — it has better autocomplete, debugging tools, and a built-in library manager. Choose your OS: Windows, macOS, or Linux.

3
Install it

Windows: Run the .exe installer. Click "Install" for all components including drivers.
macOS: Open the .dmg and drag Arduino IDE to Applications.
Linux: Extract the .zip and run install.sh as root.

4
Install USB Drivers (Windows only)

If you have a clone Arduino (CH340 chip), you need to install the CH340 driver separately. Download from: wch-ic.com/downloads/CH341SER_ZIP.html. After installing, restart your computer.

5
Launch the IDE

Open Arduino IDE. You'll see the editor window with a toolbar at the top and a console at the bottom.

Connecting Your Arduino

1
Plug in via USB

Connect your Arduino Uno to your computer using the USB-A to USB-B cable. The board's power LED (labeled PWR) should light up green.

2
Select the Board

In the IDE: go to Tools → Board → Arduino AVR Boards → Arduino Uno

3
Select the Port

Go to Tools → Port. On Windows it'll be something like COM3. On Mac/Linux it'll be /dev/cu.usbmodem... or /dev/ttyUSB0. Select the one that appeared after you plugged in the Arduino.

⚠️
Port Not Showing? If no port appears: (1) Try a different USB cable — many phone charging cables don't carry data. (2) Install/reinstall CH340 drivers if using a clone. (3) Check Device Manager (Windows) or System Information (Mac) for errors.

🔍 Lesson 3: Exploring the Arduino Uno Board

Before writing code, let's understand every part of the Arduino Uno. Knowing your hardware is just as important as knowing the code.

Arduino Uno R3 — Board Overview

Key Parts Explained

PartDescription
ATmega328PThe brain — microcontroller that runs your code. 32KB Flash, 2KB SRAM, 1KB EEPROM.
Digital Pins 0–13Can be INPUT or OUTPUT. Pins 3,5,6,9,10,11 marked with ~ support PWM (fading).
Analog Pins A0–A5Read analog voltages (0–5V) as values 0–1023. Can also work as digital pins.
5V & 3.3V pinsPower output pins for your components. Never exceed these voltages!
GND pinsGround (negative terminal). Every circuit needs a GND connection.
VinExternal power input (7–12V) — bypasses USB power.
Reset ButtonRestarts your sketch from the beginning.
TX/RX LEDsBlink when data is being sent/received via USB — useful for debugging.
L LED (Pin 13)Built-in LED connected to digital pin 13 — perfect for your first test!
ICSP HeaderIn-circuit Serial Programming — used for programming without USB.
🚨
Important Safety Rules Never connect more than 5V to digital/analog pins. Never short 5V to GND. Never draw more than 40mA from a single pin (200mA total). Always use resistors with LEDs!

💡 Lesson 4: Your First Sketch – Blink!

Every programmer's "Hello World" — in Arduino, it's the Blink sketch. We'll make the built-in LED on the board blink on and off. No extra hardware needed!

The Blink Code

Arduino C++ 
// Blink — Your First Arduino Sketch!
// This blinks the built-in LED on Pin 13

#define LED_PIN 13   // The built-in LED is on pin 13

// setup() runs ONCE when the board powers on
void setup() {
  pinMode(LED_PIN, OUTPUT);  // Set pin 13 as an output
}

// loop() runs FOREVER after setup()
void loop() {
  digitalWrite(LED_PIN, HIGH);  // Turn LED ON  (5V to pin)
  delay(1000);                  // Wait 1000ms = 1 second
  digitalWrite(LED_PIN, LOW);   // Turn LED OFF (0V to pin)
  delay(1000);                  // Wait 1 second
}

How to Upload

1
Open a New Sketch

In Arduino IDE: File → New Sketch. Delete any default code.

2
Type or Paste the Code

Copy the code above and paste it into the IDE.

3
Verify First

Click the ✅ Verify button (or Ctrl+R). Wait for "Compilation complete" in the console. If there are errors, check for typos.

4
Upload

Click the ⬆️ Upload button (or Ctrl+U). The TX/RX LEDs will blink briefly. Wait for "Upload complete."

5
Watch It Blink!

The orange LED labeled "L" on your Arduino should now blink: 1 second ON, 1 second OFF. Congratulations — you just programmed your first Arduino! 🎉

🎯
Try It! Change the delay values. Make it blink faster (try delay(200)) or make an SOS pattern (3 short, 3 long, 3 short). Experimenting is the best way to learn!

📖 Lesson 5: Understanding the Code Structure

Now let's break down what every part of the Arduino code actually means. This structure applies to every single Arduino sketch you'll ever write.

The Two Required Functions

Structure 
// 1. GLOBAL AREA — Variables and constants declared here
//    are available everywhere in your code
int myVariable = 10;

// 2. setup() — Runs ONCE when the board starts
//    Use it to configure pins, start Serial, initialize sensors
void setup() {
  // Your initialization code here
}

// 3. loop() — Runs FOREVER in a continuous cycle
//    This is your main program logic
void loop() {
  // Your main code here — repeats forever
}

Comments

Arduino C++ 
// This is a single-line comment — Arduino ignores it

/*
  This is a multi-line comment.
  Use it to explain longer sections of code.
  Everything between the asterisks is ignored.
*/

void setup() {
  // Comments make your code readable — use them always!
}

Core Arduino Functions You'll Use Constantly

Arduino C++ 
// === PIN CONTROL ===
pinMode(pin, MODE);           // MODE = INPUT, OUTPUT, or INPUT_PULLUP
digitalWrite(pin, value);     // value = HIGH (5V) or LOW (0V)
int val = digitalRead(pin);  // Returns HIGH or LOW
analogWrite(pin, value);      // PWM: value 0–255 (only ~ pins)
int raw = analogRead(pin);   // Read analog: 0–1023 (A0–A5)

// === TIME ===
delay(ms);                    // Pause for milliseconds
delayMicroseconds(us);        // Pause for microseconds
unsigned long t = millis();  // Time since board started (ms)
unsigned long u = micros();  // Time since board started (μs)

// === SERIAL (debugging) ===
Serial.begin(9600);           // Start serial at 9600 baud
Serial.print("Hello");        // Print without newline
Serial.println("World");      // Print with newline
Serial.println(variable);     // Print any variable

// === MATH ===
map(val, inMin, inMax, outMin, outMax); // Scale a value
constrain(val, min, max);     // Keep value within range
abs(x);                       // Absolute value
random(min, max);             // Random number
📝
Remember Arduino code is a simplified version of C++. You don't need to know C++ deeply, but understanding the basics (variables, if statements, loops, functions) will make you a much better Arduino programmer. These are all taught through the projects in upcoming modules!
