Motors & Movement

Make things move! This module covers servo motors for precise positioning, DC motors for wheels and fans, and stepper motors for exact rotation. By the end you'll build a working mini robot car.

⏱️ ~2.5 hours
📗 5 lessons
🟡 Intermediate

⚙️ Lesson 1: Servo Motor Basics

A servo motor is a special motor that rotates to a specific angle on command. It has built-in position control — you tell it "go to 90 degrees" and it goes there precisely. This makes servos perfect for robot arms, steering, camera mounts, and doors.

Inside a Servo

A servo contains a small DC motor, a gearbox (for torque), a potentiometer (to sense position), and a control circuit. The 3-wire interface (power, ground, signal) makes it easy to connect.

Servo WireColorArduino Connection
SignalOrange or YellowPin 9
Power (VCC)Red5V
Ground (GND)Brown or BlackGND
⚠️
Power Warning: For a single small SG90 servo, powering from Arduino's 5V pin is fine. For larger servos or multiple servos, use an external power supply (like a 6V battery pack) — they draw too much current and will reset your Arduino.

Basic Servo Control

Arduino C++
#include <Servo.h>  // Built-in library — no installation needed

Servo myServo;

void setup() {
  myServo.attach(9);  // Connect servo to pin 9
}

void loop() {
  // Sweep from 0 to 180 degrees
  for (int angle = 0; angle <= 180; angle++) {
    myServo.write(angle);
    delay(10);
  }
  // Sweep back from 180 to 0
  for (int angle = 180; angle >= 0; angle--) {
    myServo.write(angle);
    delay(10);
  }
}

Potentiometer-Controlled Servo

Arduino C++
#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(9);
}

void loop() {
  int potVal = analogRead(A0);    // 0 to 1023
  int angle  = map(potVal, 0, 1023, 0, 180); // Scale to 0–180°
  myServo.write(angle);
  delay(15);  // Give servo time to reach position
}

🔋 Lesson 2: DC Motors & L298N Driver

A DC motor spins continuously in one direction (or the other). Unlike servos, they have no built-in position sensing. You control their direction and speed. DC motors are used in wheels, fans, pumps, and conveyor belts.

Never connect a DC motor directly to an Arduino pin! Motors draw far more current than the Arduino can supply (200mA max per pin). You need a motor driver IC like the L298N.

DC Motor Control Code

Arduino C++
// L298N Motor Driver — Single Motor
#define ENA 9   // Speed (PWM)
#define IN1 2
#define IN2 3

void motorForward(int speed) {
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  analogWrite(ENA, speed);  // 0 = stop, 255 = full speed
}

void motorBackward(int speed) {
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, HIGH);
  analogWrite(ENA, speed);
}

void motorStop() {
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, LOW);
  analogWrite(ENA, 0);
}

void setup() {
  pinMode(ENA, OUTPUT);
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
}

void loop() {
  motorForward(200);  delay(2000);  // Forward 2 seconds
  motorStop();        delay(500);   // Stop 0.5s
  motorBackward(150); delay(2000);  // Backward 2 seconds
  motorStop();        delay(500);
}

🎯 Lesson 3: Stepper Motors

A stepper motor moves in precise, discrete steps — each step is an exact angle (often 1.8° per step, giving 200 steps per revolution). This makes stepper motors ideal for 3D printers, CNC machines, and camera sliders where exact positioning is critical.

Types of Stepper Motors

TypeExampleSteps/RevUse Case
Unipolar28BYJ-48 + ULN20032048 (half-step)Beginner projects, small loads
BipolarNEMA17 + A4988200 (1.8°/step)3D printers, CNC

28BYJ-48 Stepper with ULN2003 Driver

ULN2003 PinArduino Pin
IN1Pin 8
IN2Pin 9
IN3Pin 10
IN4Pin 11
VCC5V
GNDGND
Arduino C++
#include <Stepper.h>

// 28BYJ-48 has 2048 steps per revolution (half-step mode)
#define STEPS_PER_REV 2048

// Create Stepper object: steps, IN1, IN3, IN2, IN4
Stepper myStepper(STEPS_PER_REV, 8, 10, 9, 11);

void setup() {
  myStepper.setSpeed(10); // RPM (10 is good for 28BYJ-48)
  Serial.begin(9600);
}

void loop() {
  Serial.println("Clockwise one revolution");
  myStepper.step(STEPS_PER_REV);  // Positive = clockwise
  delay(1000);

  Serial.println("Counter-clockwise one revolution");
  myStepper.step(-STEPS_PER_REV); // Negative = counter-clockwise
  delay(1000);
}

🎛️ Lesson 4: Motor Speed Control

Controlling motor speed precisely is key for smooth robot movement, conveyor belts, and fans. We use PWM (analogWrite) for DC motors and the enable pin (EN) on the L298N to vary speed from 0 to 255.

Speed Ramping — Smooth Acceleration

Jumping instantly to full speed strains motors and mechanical parts. Ramping smoothly is better:

Arduino C++
// Smooth acceleration and deceleration
void rampUp() {
  for (int speed = 0; speed <= 255; speed += 5) {
    motorForward(speed);
    delay(20);
  }
}

void rampDown() {
  for (int speed = 255; speed >= 0; speed -= 5) {
    motorForward(speed);
    delay(20);
  }
}

Potentiometer Speed Control

Arduino C++
void loop() {
  int potVal = analogRead(A0);               // 0–1023
  int speed   = map(potVal, 0, 1023, 0, 255); // Scale
  int minSpeed = 80;                          // Min speed to overcome friction

  if (speed < minSpeed) {
    motorStop();
  } else {
    motorForward(speed);
  }
  delay(10);
}

🤖 Lesson 5: Mini Robot Car Project

🚗 Build: Two-Wheel Drive Robot Car

Build a 2-wheel drive robot car controlled over Serial. This project combines everything from Modules 3–6.

2x DC Motors L298N Driver Robot Car Chassis Kit Arduino Uno 7.4V LiPo or 4x AA Battery
Arduino C++
// Robot Car — Serial Controlled
// Send W/A/S/D from Serial Monitor to drive

// Motor A (Left)
#define ENA 9
#define IN1 2
#define IN2 3
// Motor B (Right)
#define ENB 10
#define IN3 4
#define IN4 5

#define SPEED 200

void forward() {
  digitalWrite(IN1,HIGH); digitalWrite(IN2,LOW); analogWrite(ENA,SPEED);
  digitalWrite(IN3,HIGH); digitalWrite(IN4,LOW); analogWrite(ENB,SPEED);
}
void backward() {
  digitalWrite(IN1,LOW); digitalWrite(IN2,HIGH); analogWrite(ENA,SPEED);
  digitalWrite(IN3,LOW); digitalWrite(IN4,HIGH); analogWrite(ENB,SPEED);
}
void turnLeft() {   // Right wheel forward, left wheel stopped
  digitalWrite(IN1,LOW); analogWrite(ENA,0);
  digitalWrite(IN3,HIGH); digitalWrite(IN4,LOW); analogWrite(ENB,SPEED);
}
void turnRight() {  // Left wheel forward, right wheel stopped
  digitalWrite(IN1,HIGH); digitalWrite(IN2,LOW); analogWrite(ENA,SPEED);
  digitalWrite(IN3,LOW); analogWrite(ENB,0);
}
void stopCar() {
  analogWrite(ENA,0); analogWrite(ENB,0);
}

void setup() {
  int pins[] = {ENA,IN1,IN2,ENB,IN3,IN4};
  for(int i=0;i<6;i++) pinMode(pins[i],OUTPUT);
  Serial.begin(9600);
  Serial.println("W=Forward S=Back A=Left D=Right X=Stop");
}

void loop() {
  if (Serial.available()) {
    char cmd = Serial.read();
    switch (cmd) {
      case 'W': case 'w': forward();   break;
      case 'S': case 's': backward();  break;
      case 'A': case 'a': turnLeft();  break;
      case 'D': case 'd': turnRight(); break;
      default:               stopCar();   break;
    }
  }
}
