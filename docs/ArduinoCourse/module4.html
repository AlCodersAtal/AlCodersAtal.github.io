Analog I/O & Sensors

The real world isn't binary. Learn to read continuously varying signals — temperature, light, and position — using Arduino's analog inputs and PWM outputs.

⏱️ ~2 hours
📗 5 lessons
🟢 Beginner

📊 Lesson 1: Analog vs Digital

Digital signals are binary — they're either HIGH (5V) or LOW (0V). But the real world has gradations. Temperature isn't just hot or cold — it's 24.6°C. Light isn't just bright or dark — it varies continuously. That's where analog comes in.

FeatureDigitalAnalog
Values0 or 1 (LOW/HIGH)Continuous range
Arduino pinsPins 0–13Pins A0–A5
Read functiondigitalRead() → 0 or 1analogRead() → 0 to 1023
ExampleButton stateTemperature, light level, rotation

The ADC (Analog-to-Digital Converter)

The Arduino Uno has a 10-bit ADC. This means it divides the 0–5V input range into 2¹⁰ = 1024 steps. So analogRead() returns a value from 0 (0V) to 1023 (5V).

Formula
Voltage = (analogRead value / 1023) × 5V

Example: analogRead() = 512
Voltage = (512 / 1023) × 5 = 2.50V (approx. half of 5V)

🎛️ Lesson 2: analogRead & Potentiometer

A potentiometer (pot) is a variable resistor with a knob. It's the perfect component to learn analogRead — turning the knob changes the voltage output from 0V to 5V.

Wiring

Potentiometer PinArduino
Left pinGND
Middle pin (wiper)A0
Right pin5V
Arduino C++
void setup() {
  Serial.begin(9600);
}

void loop() {
  int raw = analogRead(A0);         // 0–1023
  float voltage = (raw / 1023.0) * 5.0; // Convert to volts

  Serial.print("Raw: ");
  Serial.print(raw);
  Serial.print("  Voltage: ");
  Serial.print(voltage);
  Serial.println(" V");
  delay(200);
}
💡
Open Serial Monitor!After uploading, go to Tools → Serial Monitor (or Ctrl+Shift+M). Set baud rate to 9600. You'll see the values updating as you turn the knob.

The map() Function

Often you want to convert the 0–1023 range to something more useful. The map() function does this elegantly.

Arduino C++
// map(value, fromLow, fromHigh, toLow, toHigh)
int raw = analogRead(A0);          // 0–1023
int percent = map(raw, 0, 1023, 0, 100); // 0–100%
int angle   = map(raw, 0, 1023, 0, 180); // 0°–180° for servo
int pwm     = map(raw, 0, 1023, 0, 255); // 0–255 for PWM

〰️ Lesson 3: PWM & analogWrite

Arduino can't output true analog voltages from digital pins — but it can fake it using PWM (Pulse Width Modulation). PWM rapidly switches a pin between HIGH and LOW so fast that components see it as an in-between voltage.

PWM — Duty Cycle Controls Perceived Brightness / Speed
Arduino C++
// Fade LED in and out using PWM
#define LED_PIN 9   // Must be a PWM pin (~)

void setup() {
  pinMode(LED_PIN, OUTPUT);
}

void loop() {
  // Fade IN
  for (int i = 0; i <= 255; i++) {
    analogWrite(LED_PIN, i);
    delay(8);
  }
  // Fade OUT
  for (int i = 255; i >= 0; i--) {
    analogWrite(LED_PIN, i);
    delay(8);
  }
}

Quick Check

analogWrite(pin, 127) produces approximately what output?

🌡️ Lesson 4: Temperature Sensor (LM35)

The LM35 is a classic precision temperature sensor. It outputs 10mV per °C, making it incredibly easy to use with analogRead.

LM35 Pinout

PinConnection
Pin 1 (left, flat side)5V (VCC)
Pin 2 (middle)A0 (analog output)
Pin 3 (right)GND
⚠️
Orientation Matters!The LM35 has a flat and a curved side. Look at it from the flat side — left is VCC, middle is output, right is GND. Installing backwards will make it get extremely hot instantly!
Arduino C++
void setup() {
  Serial.begin(9600);
}

void loop() {
  int raw = analogRead(A0);

  // LM35: 10mV per °C
  // voltage = (raw / 1023.0) * 5000 mV
  // temperature = voltage / 10
  float voltage = (raw / 1023.0) * 5000.0;  // in mV
  float tempC = voltage / 10.0;             // in Celsius
  float tempF = (tempC * 9.0 / 5.0) + 32.0; // in Fahrenheit

  Serial.print("Temp: ");
  Serial.print(tempC);
  Serial.print(" °C  | ");
  Serial.print(tempF);
  Serial.println(" °F");
  delay(1000);
}
📝
DHT11/DHT22 AlternativeFor projects requiring humidity as well, the DHT11 and DHT22 sensors are popular choices. They use a digital interface and require the DHT sensor library (install via Library Manager). They are more accurate than the LM35 but require a library to use.

🌙 Lesson 5: Project – Automatic Night Light (LDR)

An LDR (Light Dependent Resistor) changes its resistance based on ambient light — it has high resistance in the dark and low resistance in the light. We'll use this to build an automatic LED night light.

🌙 Automatic Night Light

The LED automatically turns on when the room gets dark, and off when it's bright.

Arduino UnoLDR (photoresistor)10kΩ ResistorLED220Ω ResistorBreadboard + Wires

Voltage Divider Circuit for LDR

An LDR alone can't connect directly to Arduino — we need a voltage divider with a 10kΩ fixed resistor. Together they form a circuit that outputs varying voltage to A0.

ConnectionArduino
5V → LDR → junction pointA0
Junction → 10kΩ → GNDGND
LED (via 220Ω)Pin 13
Arduino C++
#define LDR_PIN  A0
#define LED_PIN  13
#define THRESHOLD 400  // Adjust based on your lighting

void setup() {
  pinMode(LED_PIN, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int lightLevel = analogRead(LDR_PIN);
  Serial.print("Light level: ");
  Serial.println(lightLevel);

  if (lightLevel < THRESHOLD) {
    digitalWrite(LED_PIN, HIGH); // Dark → LED ON
    Serial.println("DARK — Light ON");
  } else {
    digitalWrite(LED_PIN, LOW);  // Bright → LED OFF
    Serial.println("BRIGHT — Light OFF");
  }
  delay(300);
}
🎯
Calibrate the ThresholdOpen Serial Monitor and cover the LDR with your hand — note the value. Then expose it to normal light — note that value. Set THRESHOLD to something in between. Every room/LDR combination is different!
