// FAQ

Common questions

Do I need any experience to start? ▾
Absolutely none. This course is built for complete beginners. If you can type and use a browser, you're ready to start.
What tools do I need? ▾
Just a computer and VS Code (free). No paid software required.
How long does the course take? ▾
At 1 hour/day, most learners finish in 4 weeks. Learn at your own pace — no deadline.
Will I build a real website at the end? ▾
Yes! By Module 8 you'll build and publish your own portfolio website — live on the internet, built entirely by you.
Is this course free? ▾
100% free. All AlCoders lessons, projects, and resources are free. Just start!
