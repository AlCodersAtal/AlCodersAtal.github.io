How to make a Gas Leakage Detector and Buzzer Alarm System

A gas leakage detector and buzzer alarm system uses a gas sensor to detect the presence of gas in the environment and triggers a buzzer alarm if the gas concentration exceeds a certain threshold. The gas sensor forms a voltage divider with a resistor, and its output voltage changes with gas concentration. This voltage is read by the Arduino through an analog pin. The Arduino compares the reading to a predefined threshold to decide whether to activate the buzzer. If the gas concentration is high (high sensor output voltage), the buzzer turns on; otherwise, it remains off. The circuit consists of a gas sensor connected to a 10kΩ resistor forming a voltage divider, with one end connected to 5V, the other to ground, and the midpoint connected to the Arduino’s analog pin. The buzzer is connected to a digital pin via a 220Ω resistor.

Here's the code:

You can copy it and paste it in the Arduino IDE.

 const int gasSensorPin = A0;  // Gas sensor connected to analog pin A0
 const int buzzerPin = 13;     // Buzzer connected to digital pin 13
 int threshold = 300;          // Threshold value to determine gas presence

  void setup() {
 pinMode(buzzerPin, OUTPUT);  // Set the buzzer pin as OUTPUT
 Serial.begin(9600);          // Start serial communication for debugging
  }
  void loop() {
  int gasValue = analogRead(gasSensorPin); // Read the value from gas sensor
  Serial.println(gasValue);                // Print the gas sensor value for debuggi
  if (gasValue > threshold) {              // If gas concentration is high
          digitalWrite(buzzerPin, HIGH);       // Turn on the buzzer
  } else {                                 // If gas concentration is low
          digitalWrite(buzzerPin, LOW);        // Turn off the buzzer

  delay(100);  // Small delay for stability
  }
}

Working:

The gas leakage detector and buzzer alarm system uses a gas sensor to detect gas levels. The gas sensor is connected to an analog pin of the Arduino, which reads the voltage across the sensor. The Arduino compares this voltage to a predefined threshold value to determine whether the gas concentration is high or low. If the gas concentration is high (high sensor output voltage), the buzzer is turned on. If the gas concentration is low (low sensor output voltage), the buzzer is turned off. This system can be used for applications like gas leakage detection in homes, industries, and more.
