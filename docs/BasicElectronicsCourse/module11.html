11.1 Buttons & Debouncing

PracticalBeginner

Buttons are the simplest input. Connect between digital pin and GND with INPUT_PULLUP — reads LOW when pressed, HIGH when released.

📌 Debouncing
Mechanical buttons bounce electrically, causing multiple triggers. Fix: after detecting change, ignore input for 50ms. Or use the Bounce2 library.

11.2 Analog Sensor Reading

PracticalBeginner

Read analog sensors like LDRs and potentiometers with analogRead() and convert to useful values with map().

💡 LDR Circuit
LDR + 10kΩ divider → A0. int light = analogRead(A0); // 0 (dark) to 1023 (bright). int pct = map(light, 0, 1023, 0, 100);
📌 map() Function
map(value, fromLow, fromHigh, toLow, toHigh) — rescales a number. map(analogRead(A0), 0, 1023, 0, 255) maps 0–5V to 0–255 for analogWrite().

11.3 I2C Communication

TheoryBeginner

I2C connects up to 128 devices with just 2 wires: SDA (A4) and SCL (A5) on Arduino Uno.

📌 Using Wire Library
#include . Wire.begin(); Wire.beginTransmission(addr); Wire.write(data); Wire.endTransmission(); Wire.requestFrom(addr, bytes); Wire.read();
💡 Common I2C Addresses
OLED 0.96": 0x3C. MPU-6050: 0x68. BMP280: 0x76. DS3231 RTC: 0x68.

11.4 LCD Display

PracticalBeginner

16×2 I2C LCD reduces wiring to 4 pins (Vcc, GND, SDA, SCL) — much simpler than direct wiring.

📌 I2C LCD Code
#include <LiquidCrystal_I2C.h>. LiquidCrystal_I2C lcd(0x27, 16, 2); lcd.init(); lcd.backlight(); lcd.setCursor(0,0); lcd.print("Hello World!");

11.5 Servo Motors

PracticalBeginner

Servo motors move to a precise angle (0–180°) on command — perfect for beginner robotics.

📌 Servo Code
#include <Servo.h>. Servo myservo; myservo.attach(9); myservo.write(90); // go to 90 degrees
💡 Pot-Controlled Servo
int pot = analogRead(A0); int angle = map(pot, 0, 1023, 0, 180); myservo.write(angle); — potentiometer controls servo position in real-time!
