6.1 What is a Transistor?

TheoryBeginner

The transistor — invented in 1947 — is a semiconductor device used as a switch or amplifier. Every smartphone contains billions.

📌 Two Main Types
BJT (Bipolar Junction Transistor): current-controlled, 3 terminals: Base, Collector, Emitter. MOSFET: voltage-controlled, 3 terminals: Gate, Drain, Source.

6.2 NPN BJT as Switch

TheoryBeginner

An NPN BJT (like BC547, 2N2222) is the most common transistor for beginner switching applications.

Ic = hFE × Ib
Collector current = current gain × base current. Typical hFE = 100–300
📌 How to Switch
Base LOW → transistor OFF. Base HIGH (via resistor) → transistor saturated (fully ON). Lets 5mA microcontroller signal control 500mA motor.
💡 Base Resistor
Always limit base current with resistor. If Vcc=5V, Ib_desired=1mA: R = (5−0.7)/0.001 = 4.3kΩ → use 4.7kΩ.

6.3 Transistor Switch — Circuit

PracticalBeginner

Circuit: Arduino pin → 1kΩ → NPN Base. Motor between 12V and Collector. Emitter to GND. Arduino HIGH → motor runs. LOW → motor stops.

⚠️ Flyback Diode
ALWAYS add a flyback diode (1N4007, cathode to Vcc) across motors, relays, and solenoids to absorb voltage spikes when switching off.

6.4 MOSFET Basics

TheoryBeginner

MOSFETs (like IRLZ44N) are voltage-controlled and draw almost zero gate current — perfect for microcontrollers.

📌 N-Channel MOSFET
Gate voltage above threshold (~2–4V for logic-level) turns it ON. Load between Drain and Vcc. Source to GND.
💡 Logic-Level MOSFETs
IRLZ44N, 2N7000 fully turn on with 5V gate signal — perfect for Arduino. Standard MOSFETs need 10V.

6.5 Transistor Amplifier Overview

TheoryBeginner

A transistor amplifies weak signals: small base current variations → large collector current variations.

📌 Common Amplifier Configurations
Common Emitter: high voltage gain, inverted signal. Used in audio amps. Emitter Follower: gain ~1, high current drive, impedance matching. Common Base: high frequency.
