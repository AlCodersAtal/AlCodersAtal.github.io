12.1 Project 1: Smart Night Light

PracticalBeginner

Build an auto-dimming LED that brightens in darkness and turns off in daylight.

📌 Components
Arduino, LDR, 10kΩ, 220Ω, LED (Pin 9 PWM), breadboard.
💡 Core Code
int ldr = analogRead(A0); int brightness = map(ldr, 200, 900, 255, 0); brightness = constrain(brightness, 0, 255); analogWrite(9, brightness);

12.2 Project 2: Motion Alarm

PracticalBeginner

PIR-triggered alarm with armed/disarmed states, buzzer, and LED indicators.

📌 Components
Arduino, PIR HC-SR501, active buzzer, red LED, green LED, push button, resistors.
💡 PIR Setup
Adjust HC-SR501's sensitivity and delay pots. Use repeat-trigger jumper mode for continuous detection while motion persists.

12.3 Project 3: Weather Station

PracticalBeginner

Display real-time temperature, humidity, and heat index on a 16×2 I2C LCD.

📌 Libraries
Install via Library Manager: DHT sensor library (Adafruit) and LiquidCrystal_I2C. float t = dht.readTemperature(); float h = dht.readHumidity();
💡 Enhancement
Add BMP280 for pressure and altitude. Send data to Serial Plotter to visualise temperature changes over time!

12.4 Project 4: Line-Following Robot

PracticalBeginner

Two-wheel robot that follows a black line using IR sensors and an L298N motor driver.

📌 Logic
Left sensor on line → turn right. Right sensor on line → turn left. Both on → go straight. Both off → stop or search.
💡 Tuning Tips
IR sensor height: 5–10mm. Line width: 3–4cm. Tune motor speeds with PWM. Debug sensor values first with Serial.println before adding motor code!

12.5 Project 5: IoT Sensor Station

PracticalBeginner

Use NodeMCU ESP8266 to send DHT22 readings to ThingSpeak cloud for web dashboard visualisation.

📌 Setup
Install ESP8266 boards in IDE. Connect to WiFi with WiFi.begin(). Get ThingSpeak API key. POST data to ThingSpeak field every 15 seconds.
