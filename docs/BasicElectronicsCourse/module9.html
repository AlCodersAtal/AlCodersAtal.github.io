9.1 Analog vs Digital

TheoryBeginner

All signals are either analog (continuous, like temperature) or digital (two states: HIGH/LOW).

📌 ADC — Analog to Digital
Arduino has a 10-bit ADC: converts 0–5V to 0–1023. Resolution = 5V/1024 ≈ 4.9mV per step.

9.2 Common Sensors

PracticalBeginner
🔩

LDR

Resistance drops with light. Use in voltage divider for light level detection.

🔩

NTC Thermistor

Resistance drops with heat. Combine with voltage divider for temperature sensing.

🔩

PIR HC-SR501

Detects human motion. Digital output HIGH when motion detected.

🔩

HC-SR04

Ultrasonic distance sensor. 2cm–400cm range. Trigger pulse → measure echo time.

🔩

DHT22

Temperature + humidity sensor. Digital one-wire protocol. More accurate than DHT11.

🔩

MPU-6050

6-axis IMU via I2C. Accelerometer + gyroscope for motion/orientation.

🔩

MQ-2

Gas sensor (LPG, smoke). Analog output. Needs warm-up time.

🔩

TCRT5000

IR reflective sensor for line following. Short range (1–10mm).

9.3 Signal Conditioning

TheoryBeginner

Raw sensor signals often need preparation before a microcontroller can use them.

📌 Techniques
Amplification (op-amp) · Low-pass filtering (RC) · Level shifting (resistor divider) · Buffering (voltage follower) · ADC conversion.

9.4 Pull-up & Pull-down Resistors

PracticalBeginner

Digital inputs must never be left floating (unconnected) — they pick up noise and read random values.

📌 Solutions
Pull-up (10kΩ to Vcc): undriven → reads HIGH. Pull-down (10kΩ to GND): undriven → reads LOW. Arduino has built-in pull-ups activated with INPUT_PULLUP.
⚠️ Voltage Levels
Arduino is 5V. Some sensors are 3.3V. Never connect 5V to a 3.3V input directly — use a voltage divider or level shifter.

9.5 Sensor Reading Preview

PracticalBeginner

Before Arduino code in Module 10, here's the pattern for reading sensors.

📌 Voltage Divider Sensor Pattern
Sensor as bottom resistor → midpoint to analog pin → analogRead() returns 0–1023 → multiply by 5.0/1023.0 for volts.
💡 Calibration
Raw ADC values are meaningless without calibration. Always test at known reference points and apply a conversion formula.
