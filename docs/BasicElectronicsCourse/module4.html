4.1 Series Circuits

TheoryBeginner

In a series circuit, components are connected end-to-end in a single path. There is only one path for current to flow.

R_total = R1 + R2 + R3 + ...
Total resistance in series adds up
📌 Key Properties of Series
Current is the same through all components. Voltage divides across each component proportionally to its resistance. If one component fails open, the entire circuit stops working.
V_total = V1 + V2 + V3
Voltages add up in series (Kirchhoff's Voltage Law)

4.2 Parallel Circuits

TheoryBeginner

In a parallel circuit, components are connected across the same two nodes, giving current multiple paths to flow.

1/R_total = 1/R1 + 1/R2 + 1/R3
Total resistance in parallel is always less than the smallest resistor
📌 Key Properties of Parallel
Voltage is the same across all parallel branches. Current divides between branches. If one branch fails, others continue working.
💡 Shortcut for Two Resistors
For just two resistors: R_total = (R1 × R2) / (R1 + R2). Much faster!

4.3 Kirchhoff's Voltage Law (KVL)

TheoryBeginner

KVL: The sum of all voltage drops around any closed loop = 0. Energy is conserved — whatever voltage a source provides, the components consume exactly that much.

ΣV = 0 around any loop
Sum of all voltages in a loop = 0
📌 How to Apply KVL
1. Choose a loop direction. 2. Start at any point. 3. Add voltage rises (+) and drops (−). 4. The total must equal zero.

4.4 Kirchhoff's Current Law (KCL)

TheoryBeginner

KCL: The total current entering a node equals the total current leaving that node. Charge cannot accumulate at a point.

ΣI_in = ΣI_out at any node
Current in = Current out
📌 Practical Use
At any junction, add all currents flowing in and all currents flowing out — they must be equal.

4.5 Voltage Divider

PracticalBeginner

A voltage divider uses two resistors in series to produce a fraction of the input voltage. One of the most used circuits in electronics.

Vout = Vin × R2 / (R1 + R2)
Output voltage is proportional to R2's share of total resistance
💡 Example
5V, R1=R2=10kΩ: Vout = 2.5V. This is exactly how potentiometers and sensor interfaces work!
