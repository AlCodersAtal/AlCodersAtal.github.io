7.1 The Breadboard

PracticalBeginner

A breadboard lets you build circuits without soldering — the ideal tool for learning and prototyping.

📌 Internal Connections
Power rails (long columns): entire column connected — use for Vcc and GND. Component rows (middle): each row of 5 holes connected. Central gap separates IC pin rows.
💡 Best Practices
Short wires, correct power rail colours (red=Vcc, black=GND), double-check before powering, connect GND before Vcc.

7.2 Reading Schematics

TheoryBeginner

A schematic is a standardised circuit diagram using symbols. Reading them is as important as writing code.

📌 Common Symbols
Wire = line. Junction = dot. Resistor = rectangle/zigzag. Capacitor = two parallel lines. Battery = alternating long/short lines. Ground = triangle/horizontal lines.

7.3 The Multimeter

PracticalBeginner

A multimeter measures voltage, current, resistance, and continuity. The most essential tool in electronics.

📌 Key Measurements
DC Voltage: probes across component. Resistance: power disconnected first! Current: probes in series. Continuity: tests if path is complete (buzzes).
⚠️ Safety
NEVER measure resistance with power on. NEVER measure current across voltage. Avoid mains AC until trained.

7.4 Power Supplies

PracticalBeginner

Stable power is essential for reliable circuits.

📌 Power Options
AA batteries (1.5V), 9V block, USB (5V stable), bench PSU (adjustable). LM7805 IC provides stable 5V from 7–35V input.
💡 LM7805 Circuit
Input capacitor (100µF) → LM7805 → Output capacitor (10µF). Add 100nF ceramics for decoupling. Use heatsink if input voltage is much higher than 5V.

7.5 First Breadboard Projects

PracticalBeginner
📌 Project 1: LED + Resistor
5V → 220Ω → LED → GND. Expected current ~14mA. LED lights brightly.
📌 Project 2: Voltage Divider
Two 10kΩ resistors in series across 5V. Measure midpoint — should read 2.5V.
💡 Project 3: Transistor Switch
Arduino pin → 1kΩ → BC547 Base. LED + 220Ω between Collector and 5V. Emitter to GND. HIGH → LED on.
