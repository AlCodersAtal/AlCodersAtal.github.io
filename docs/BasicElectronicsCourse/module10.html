10.1 What is Arduino?

TheoryBeginner

Arduino is an open-source microcontroller platform that makes programming hardware accessible to beginners. Created in 2005, it has millions of users worldwide.

🔩

Digital Pins 0–13

Read/write HIGH/LOW. Pins 3,5,6,9,10,11 support PWM.

🔩

Analog Pins A0–A5

10-bit ADC. 0–5V → 0–1023. Can also be used as digital.

🔩

Power Pins

5V out, 3.3V out, GND, Vin. Max 500mA from 5V when USB powered.

🔩

ATmega328P

8-bit 16MHz CPU. 32KB Flash. 2KB SRAM. The brain of the Arduino.

10.2 IDE Setup & Sketch Structure

PracticalBeginner

The Arduino IDE is free at arduino.cc. Install, select board (Arduino Uno) and port, and upload.

📌 Every Sketch Has Two Functions
setup(): runs once on startup. Use for initialization. loop(): runs forever after setup. Put your main logic here.

10.3 Digital I/O & Blink

PracticalBeginner

Blink is the Hello World of Arduino — flash the LED on pin 13 (built-in LED on Arduino Uno).

💡 Blink Code
void setup() { pinMode(13, OUTPUT); } void loop() { digitalWrite(13, HIGH); delay(1000); digitalWrite(13, LOW); delay(1000); }
📌 Key Functions
pinMode(pin, mode): INPUT or OUTPUT. digitalWrite(pin, val): HIGH or LOW. digitalRead(pin): returns HIGH or LOW. delay(ms): pause execution.

10.4 Analog I/O & PWM

PracticalBeginner

Arduino reads 0–5V as 0–1023 via analogRead(). It simulates analog output with PWM (Pulse Width Modulation) via analogWrite().

📌 analogRead & analogWrite
int val = analogRead(A0); // 0–1023. float v = val * 5.0 / 1023.0; // convert to volts. analogWrite(9, 128); // ~2.5V average on PWM pin.
💡 PWM Explained
PWM switches pin HIGH/LOW rapidly. Duty cycle = fraction of time HIGH. 50% duty = ~2.5V average. LED or motor averages this and behaves accordingly.

10.5 Serial Monitor

PracticalBeginner

The Serial Monitor is your window into the Arduino — print debug messages, sensor values, and program state to your computer.

📌 Serial Functions
Serial.begin(9600) in setup. Serial.println(value) prints with newline. Serial.print() without newline. Open Monitor: magnifying glass icon. Set baud to match Serial.begin().
💡 Debug Strategy
Print variable values at key points in your code — this is how every programmer debugs. Add Serial.println(sensor_value) to verify sensors are reading correctly before adding complex logic.
